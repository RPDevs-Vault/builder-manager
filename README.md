# 📦 container-manager

Central hub for managing GitHub Container Registry (GHCR) assets for the **Dick-s-Vault** organization.

## 🚀 Purpose
- **OCI Hosting:** Manage Docker/OCI images hosted on `ghcr.io`.
- **Build Infrastructure:** Coordinate build artifacts from projects like `vlc-live-555`.
- **Lifecycle Management:** Automate the pruning and versioning of container images.
