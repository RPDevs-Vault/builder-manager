# Soltrade | Soltrade

### 

[](https://wons-organization-2.gitbook.io/soltrade/#overview)

Overview

The Soltrade is a powerful API for swapping in Solana chain, we take a 0.3% fee on every successful transaction you made.

### 

[](https://wons-organization-2.gitbook.io/soltrade/#supported-dexes)

Supported DEXes

• Jupiter Studio • Moonshot • Pumpfun • PumpSwap • Raydium • Raydium CPMM • Raydium Launchlab • Orca • Meteora • Jupiter • Boop • Orca • Meteora Dynamic Bonding Curve • Meteora DAMM V2 • Believe • Bonk • And more.

### 

[](https://wons-organization-2.gitbook.io/soltrade/#api-reference)

API Reference

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#base-url)

Base URL

Copy

    https://soltrade.up.railway.app

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#endpoints)

Endpoints

**POST /swap**

Execute a buy or sell trade operation.

**Request Headers:**

Copy

    Content-Type: application/json

**Request Body:**

Parameter

Type

Required

Description

`publicKey`

string

Yes

Your wallet public key

`action`

string

Yes

"buy" or "sell"

`mint`

string

Yes

Token contract address

`amount`

number

Yes

Amount of SOL or tokens to trade

`slippage`

number

Yes

Slippage percentage (0-100)

`priorityFee`

number

Yes

Priority fee amount

**Success Response:**

Copy

    {
      "txn": "BASE64_TRANSACTION_STRING"
    }

**Error Response:**

Copy

    {
      "error": "Swap failed",
      "message": "Unable to process the swap request at this time. Please try again later."
    }

### 

[](https://wons-organization-2.gitbook.io/soltrade/#usage-examples)

Usage Examples

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#buy-operation-sol-token)

Buy Operation (SOL → Token)

Copy

    curl -X POST https://soltrade.up.railway.app//swap \
      -H "Content-Type: application/json" \
      -d '{
        "publicKey": "arsc4jbDnzaqcCLByyGo7fg7S2SmcFsWUzQuDtLZh2y",
        "action": "buy",
        "mint": "4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R",
        "amount": 1,
        "slippage": 15,
        "priorityFee": 0.0005
      }'

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#sell-operation-token-sol)

Sell Operation (Token → SOL)

Copy

    curl -X POST https://soltrade.up.railway.app/swap \
      -H "Content-Type: application/json" \
      -d '{
        "publicKey": "arsc4jbDnzaqcCLByyGo7fg7S2SmcFsWUzQuDtLZh2y",
        "action": "sell",
        "mint": "4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R",
        "amount": "50%",
        "slippage": 15,
        "priorityFee": 0.0005
      }'

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#javascript-example)

JavaScript Example

Copy

    const axios = require('axios');
    
    const swapRequest = {
      publicKey: "AWPmgAr6xDJUhCa2Wxna5gd14UT98hicTXdow9XkYnNX",
      action: "buy",
      mint: "4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R",
      amount: 1,
      slippage: 15,
      priorityFee: 0.0005
    };
    
    try {
      const response = await axios.post('https://soltrade.up.railway.app/swap', swapRequest);
      console.log('Transaction:', response.data.txn);
    } catch (error) {
      console.error('Error:', error.response.data);
    }

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#python-example)

Python Example

Copy

    import requests
    import json
    
    url = "https://soltrade.up.railway.app/swap"
    data = {
        "publicKey": "AWPmgAr6xDJUhCa2Wxna5gd14UT98hicTXdow9XkYnNX ",
        "action": "buy",
        "mint": "4k3Dyjzvzp8eMZWUXbBCjEvwSkkk59S5iCNLY3QrkX6R",
        "amount": 1,
        "slippage": 15,
        "priorityFee": 0.0005
    }
    
    response = requests.post(url, json=data)
    if response.status_code == 200:
        print(f"Transaction: {response.json()['txn']}")
    else:
        print(f"Error: {response.json()}")

#### 

[](https://wons-organization-2.gitbook.io/soltrade/#rate-limit-response)

Rate Limit Response

When rate limit is exceeded, the API returns:

Copy

    {
      "error": "Rate limit exceeded",
      "message": "Too many requests. Maximum 2 requests per second allowed.",
      "retryAfter": 1
    }
