# 代币特征 | Matrixdock docs

由99.99%纯度的 LBMA（伦敦金银市场协会）规格实物金条全额支持的黄金代币。每个 XAUm 代币锚定一金衡盎司（toz）的实物金条。

可转让性和要求[](#ke-zhuan-rang-xing-he-yao-qiu)

*   将 XAUm代币打包/解包到NFT 中是无需许可的
    

*   XAUm 的初级铸造和兑换为 USDC 和美元法定货币需要 KYC
    

代币和 NFT 形式[](#dai-bi-he-nft-xing-shi)

XAUm 有 ERC-20/BEP-20 代币形式和 ERC-721/BEP-721 NFT 形式。在 ERC-20/BEP-20 代币形式中，1 XAUm 由 1 个金衡盎司 LBMA 黄金支持。在 ERC-721/BEP-721 NFT 形式中，1 个 NFT 由特定的整个 LBMA 金条支持。这种多形式的功能满足了不同用户对灵活性和增强安全性的需求。两种形式之间的转换是通过 Matrixdock Web 应用程序上的小部件启用的，其中所需的 ERC-20/BEP-20 代币数量对应于所选柱的精细权重。

分配查找[](#fen-pei-cha-zhao)

XAUm 代币和 NFT 持有者都可以使用 Matrixdock Gold 网站上的分配查找工具来查看与其代币或 NFT 对应的特定物理金条的详细信息。这种关系始终保持透明、安全和可验证。 XAUm 代币动态分配给金条。代币持有者可以检查其最后更新的动态分配，该分配可能会随着时间的推移而发生变化。为了优化不同代币持有者的全部或部分所有权的配置，代币可能会不时地重新分配到不同的物理金条。这是通过在铸币、赎回或转账发生时近乎实时地分析合约事件日志来完成的。 Matrixdock Gold NFT 直接代表对特定金条的固定分配。这确保了资产仅属于 NFT 持有者，无法被他人赎回。 用户可以通过我们的 Web 应用程序中的打包/解包功能在 XAUm 代币和 Matrixdock Gold NFT 之间进行转换。所需的 XAUm 数量将与所选金条的金衡盎司细重相对应。

破产隔离[](#po-chan-ge-li)

XAUm项目提供100%破产隔离，为投资者提供更加安全可靠的投资选择。为了实现这一里程碑，该项目设立了一个特定目的信托，该信托在法律上拥有XAUm发行人，且实现了破产隔离结构。

该目的信托将作为XAUm资产账户持有人，专门致力于持有XAUm发行者的份额。通过使用这种独特的法律架构，XAUm项目有效地减轻了与Matrixport Group相关的对手方风险，并显著增强了XAUm项目的安全性。

1.  使用稳定币铸造 XAUm 代币：您可以使用支持的稳定币（例如 USDC 和 USDT）直接从 Matrixdock 购买 XAUm。 Matrixdock 将执行物理金条的采购并将代币发行到您的钱包。 **铸造流程可参考** [铸造及赎回](/matrixdock-docs/huang-jin-dai-bi-xaum/zhu-zao-ji-shu-hui)
    
2.  将现有 LBMA 黄金代币化：作为在我们的合作伙伴金库中拥有账户的现有资产所有者，可以将经过认证的 LBMA 黄金从您的账户转移到 Matrixdock 以进行代币化。这可能需要接受质量和尽职调查。
    

XAUm 可以赎回为什么资产？[](#xaum-ke-yi-shu-hui-wei-shen-me-zi-chan)

1.  赎回成稳定币：您可以直接通过 Matrixdock 将 XAUm 兑换成受支持的稳定币，例如 USDC 和 USDT。 Matrixdock 将执行实体金条的销售并将稳定币转移到您的钱包中。 **流程可参考** [铸造及赎回](/matrixdock-docs/huang-jin-dai-bi-xaum/zhu-zao-ji-shu-hui)
    
2.  赎回为实物 LBMA 黄金：用户将需要转移足够数量的 XAUm 代币，以支付金衡盎司重量的金条加上赎回费用。可兑换的金条将根据要求提供，并视库存情况而定。
    

收取什么费用？[](#shou-qu-shen-me-fei-yong)

*   **铸币费用**
    

在锁定总价值 (TVL) 达到 1 亿美元之前，铸币费用将被免除。 [XAUm 限量优惠：免除铸币费！](/matrixdock-docs/gong-gao/qi-ta-gong-gao/xaum-xian-liang-you-hui-mian-chu-zhu-bi-fei)

# 为什么我看不到钱包 | Matrixdock docs

### 

[](#id-2)

1.浏览器插件被禁用

对于桌面用户，如果浏览器插件被禁用，他们将无法看到自己的钱包。

*   检查浏览器插件设置。如果您是 Chrome 用户，访问 chrome://extensions/
    
*   确保切换按钮为蓝色，启用插件。
    
*   启用后，您将在浏览器工具栏中看到钱包图标。访问钱包前请刷新页面。
    

### 

[](#id-2.-yi-an-zhuang-duo-ge-qian-bao)

2.已安装多个钱包

通常，当用户在浏览器中安装了多个钱包时，dapp 用来检测钱包的空间可能会发生冲突。因此，在某些情况下可能只有一个钱包可用。为了解决这个问题：

*   转到浏览器的插件设置（在 Chrome 浏览器中是：[chrome://extensions/](chrome://extensions/)
    )
    
*   确保禁用所有其他浏览器钱包（或确保它们未被设置为默认 web3 钱包），并启用目标钱包。
    
*   在访问所需钱包之前刷新浏览器。
    

此外，如果您碰巧设置了默认浏览器钱包，那么您想要的钱包可能无法显示。您可以在钱包设置中更新默认值，或者如上所述禁用插件。

### 

[](#id-3.-outdated-browser-or-extension)

3.浏览器或插件程序过时

如果桌面用户的浏览器或插件程序已经过时，他们可能无法看到嵌入浏览器的基于以太坊的 web3 钱包。MetaMask 等钱包会经常更新软件，以提高安全性和功能性。如果用户的浏览器或钱包插件已经过时，他们在访问钱包时可能会遇到问题。
