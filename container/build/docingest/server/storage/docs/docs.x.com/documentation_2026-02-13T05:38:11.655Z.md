Pricing
---------------------------------------------------

The X API uses **pay-per-usage** pricing. No monthly subscriptions — pay only for what you use.

[Pay-per-usage\
-------------\
\
Flexible, credit-based pricing. Scale up or down based on your needs with no commitments or minimum spend.](https://docs.x.com/x-api/getting-started/pricing)
[Enterprise\
----------\
\
Dedicated support, complete streams, and custom integrations for high-volume needs.](https://docs.x.com/forms/enterprise-api-interest)

Earn free [xAI API](https://docs.x.ai/)
 credits when you purchase X API credits—up to 20% back based on your spend. [Learn more](https://docs.x.com/x-api/getting-started/pricing#free-xai-api-credits)
