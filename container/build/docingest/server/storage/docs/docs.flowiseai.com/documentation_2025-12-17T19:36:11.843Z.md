# Introduction | FlowiseAI

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252FK5NWsHkLAelZq9sBlY8x%252FFlowiseIntro.gif%3Falt%3Dmedia%26token%3Dea75ba7b-32fa-447d-8872-41fe5578fe1f&width=768&dpr=4&quality=100&sign=35a65986&sv=2)

Flowise is an open source generative AI development platform for building AI Agents and LLM workflows.

It offers a complete solution that includes:

*   Visual Builder
    
*   Tracing & Analytics
    
*   Evaluations
    
*   Human in the Loop
    
*   API, CLI, SDK, Embedded Chatbot
    
*   Teams & Workspaces
    

There are 3 main visual builders namely:

*   Assistant
    
*   Chatflow
    
*   Agentflow
    

[](https://docs.flowiseai.com/#assistant)

Assistant


--------------------------------------------------------

Assistant is the most begineer-friendly way of creating an AI Agent. Users can create chat assistant that is able to follow instructions, use tools when neccessary, and retrieve knowledge base from uploaded files ([RAG](https://en.wikipedia.org/wiki/Retrieval-augmented_generation)
) to respond to user queries.

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252Fo43CuYoNE9AEaYJP1yme%252Fimage.png%3Falt%3Dmedia%26token%3D54c63256-7b14-4861-8ab4-824f774d0b8d&width=768&dpr=4&quality=100&sign=b7079b6b&sv=2)![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252F9el6yIQeSENvCV5tU3rW%252FScreenshot%25202025-06-10%2520232758.png%3Falt%3Dmedia%26token%3De14343d0-6128-45ed-a9d9-e0c934c63344&width=768&dpr=4&quality=100&sign=16ad8717&sv=2)

[](https://docs.flowiseai.com/#chatflow)

Chatflow


------------------------------------------------------

Chatflow is designed to build single-agent systems, chatbots and simple LLM flows. It is more flexible than Assistant. Users can use advance techniques like Graph RAG, Reranker, Retriever, etc.

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252FF9wMU4Dx7pwV8BiwaYPl%252Fscreely-1749593961545.png%3Falt%3Dmedia%26token%3D9138d14e-cbc0-46de-a30a-1b7ef9eaa157&width=768&dpr=4&quality=100&sign=548b5591&sv=2)![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252FhzSnmLUOcCBAlpolUcZZ%252Fscreely-1749594035877.png%3Falt%3Dmedia%26token%3D7659c389-7037-497e-b8e9-f5dc72221cae&width=768&dpr=4&quality=100&sign=ec407cd6&sv=2)

[](https://docs.flowiseai.com/#agentflow)

Agentflow


--------------------------------------------------------

Agentflow is the superset of Chatflow & Assistant. It can be used to create chat assistant, single-agent system, multi-agent systems, and complex workflow orchestration. Learn more [Agentflow V2](https://docs.flowiseai.com/using-flowise/agentflowv2)

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252Fc4EwQgQRnRYD9lbblfpE%252Fscreely-1749594614881.png%3Falt%3Dmedia%26token%3Dcd4f1a8f-e7d7-4552-afe4-eb583f5fc199&width=768&dpr=4&quality=100&sign=35901410&sv=2)![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F823733684-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F00tYLwhz5RyR7fJEhrWy%252Fuploads%252FZxGswbXpJ3wcJyNHRVSg%252Fscreely-1749594631028.png%3Falt%3Dmedia%26token%3D490115f1-562f-4300-81a7-4e3cefa52ec2&width=768&dpr=4&quality=100&sign=e6c44272&sv=2)

# Introduction | Español | FlowiseAI

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252FXxve37yNH63uTNoBYJ1Y%252FFlowise%2520Cropped%2520White%2520High%2520Res.png%3Falt%3Dmedia%26token%3De022b6af-b237-4724-9e4c-7b895cc3cdbe&width=768&dpr=4&quality=100&sign=8f7e6694&sv=2)![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252Fkc0ui4v8wWvxDhPJEni3%252FFlowise%2520Logo%2520Cropped%2520White%2520High%2520Res.png%3Falt%3Dmedia%26token%3D597f761e-d481-4726-afff-1d916064b926&width=768&dpr=4&quality=100&sign=c55049fa&sv=2)

**Flowise** es una plataforma de desarrollo de IA generativa de código abierto para construir Agentes de IA y flujos de trabajo con modelos de lenguaje (LLM).

Ofrece una solución completa que incluye:

*   Constructor visual
    
*   Trazabilidad y análisis
    
*   Evaluaciones
    
*   Intervención humana (Human in the Loop)
    
*   API, SDK y chatbot embebido
    
*   Equipos y espacios de trabajo
    

Existen tres constructores visuales principales:

*   Assistant
    
*   Chatflow
    
*   Agentflow
    

[](https://docs.flowiseai.com/espanol#assistant)

Assistant


---------------------------------------------------------------

Assistant es la forma más amigable para principiantes de crear un Agente de IA. Los usuarios pueden crear un asistente conversacional capaz de seguir instrucciones, utilizar herramientas cuando sea necesario y recuperar información desde archivos subidos (RAG) para responder a consultas del usuario.

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252FUCRl3mnWvpBDkxWvYLq1%252Fimage.png%3Falt%3Dmedia%26token%3D9dc8e9e3-fc92-4571-8518-0dedb1af4815&width=768&dpr=4&quality=100&sign=83818f82&sv=2)![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252FdMb6sAtnFnhbXb1GtFT8%252FScreenshot%25202025-05-12%2520215934.png%3Falt%3Dmedia%26token%3D04a10a0e-bea1-490a-8046-714a343e8fd4&width=768&dpr=4&quality=100&sign=feaec524&sv=2)

[](https://docs.flowiseai.com/espanol#chatflow)

Chatflow


-------------------------------------------------------------

Chatflow está diseñado para construir sistemas de agente único, chatbots y flujos simples con modelos de lenguaje (LLM). Es más flexible que Assistant. Los usuarios pueden utilizar técnicas avanzadas como Graph RAG, Reranker, código personalizado, entre otros.

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252FYIa0JfQIXaF8C7zjByel%252Fwhite.png%3Falt%3Dmedia%26token%3D4c3bdbee-e73c-4f8c-80ba-0d3a3ee6c53d&width=768&dpr=4&quality=100&sign=37e06946&sv=2)![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252FGB0Idz70ftD4RRPU6zIv%252Fdark.png%3Falt%3Dmedia%26token%3Dcc5890e2-c9d3-492e-9287-e65f16efa2e7&width=768&dpr=4&quality=100&sign=2d260e55&sv=2)

[](https://docs.flowiseai.com/espanol#agentflow)

Agentflow


---------------------------------------------------------------

Agentflow es el superconjunto de Chatflow y Assistant. Se puede utilizar para crear asistentes conversacionales, sistemas de agente único, sistemas multi-agente y orquestaciones de flujo de trabajo complejas.

![](https://docs.flowiseai.com/~gitbook/image?url=https%3A%2F%2F4068692976-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FUiD7nOmFRK805sNuiieJ%252Fuploads%252F0S6K7NTpeGKhS1EPmT1r%252FFlowiseIntro.gif%3Falt%3Dmedia%26token%3D5e076af8-2db1-433c-a0e1-53fddfa3319b&width=768&dpr=4&quality=100&sign=bbdf27e0&sv=2)
