# Before Starting | LuaProt

triangle-exclamation

[hashtag](https://docs.luaprot.net/luaprot/api-documentation/before-starting#your-luaprot-api-is-not-public-which-means-do-not-share-it)

Your LuaProt API is not public, which means DO NOT SHARE IT.


----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

circle-exclamation

You have to whitelist your IP, which makes these requests on the dashboard, otherwise your requests get rejected.

[hashtag](https://docs.luaprot.net/luaprot/api-documentation/before-starting#request-headers)

Request Headers


------------------------------------------------------------------------------------------------------------------

Always include these following headers in every API request you make.

Copy

    Content-Type: application/json
    Authorization: YOUR_API_KEY

#### 

[hashtag](https://docs.luaprot.net/luaprot/api-documentation/before-starting#good-to-know)

Good To Know

*   You can make a total of **20 requests per 10 seconds** and **60 requests per minute**. Exceeding this limit will result a `429` Http Error Code. (applies for most of the routes)
    

_Now that you know this, you can start looking at the API endpoints in the next pages._

# Fetch Keys | LuaProt

`**GET**` `/api/v1/keys/fetch`

#### 

[hashtag](https://docs.luaprot.net/luaprot/api-documentation/fetch-keys#query)

Query

Query

Description

hub

HubId to look for. Default will look for every Hub.

discordId

Discord ID to look for. Default will look for every discord ID.

key

Key to look for. Default will look for every key.

hwid

HWID to look for. Default will look for every HWID.

blacklisted

Key should be blacklisted. (boolean) Default will not filter by blacklisted.

expired

Key should be expired. (boolean) Default will not filter by expired.

unassigned

Key should be unassigned. (boolean) Default will not filter by unassigned.

active

Key should be active. (boolean) Default will not filter by active.

page

Page of the query. Default is 1.

#### 

[hashtag](https://docs.luaprot.net/luaprot/api-documentation/fetch-keys#reponse)

Reponse

200

Copy

    {
          success: true,
          message: "Successfully fetched keys.",
          keys: [...],
          limitPerPage: 50,
          totalPages: 1,
          totalKeys: 1,
     }
