### Secure Mode[​](https://docs.sirius.menu/community/arrayfield/configuration/booting-library#secure-mode "Direct link to Secure Mode")

CautionWarning

If the game you're trying to run ArrayField Interface Suite on, is detecting or crashing when you use ArrayField Interface Suite, try using Secure Mode:

*   Place `getgenv().SecureMode = true` above the initial ArrayField loadstring

ArrayField will now use Secure Mode and attempt to reduce detection

Notenote

This may cause some elements of the UI to look lower quality or may increase loading times slightly

### Enabling Configuration Saving[​](https://docs.sirius.menu/community/arrayfield/configuration/booting-library#enabling-configuration-saving "Direct link to Enabling Configuration Saving")

Tiptip

1.  Enable ConfigurationSaving in the CreateWindow function
2.  Choose an appropiate FileName in the CreateWindow function
3.  Choose an unique flag identifier for each supported element you create
4.  Place `ArrayField:LoadConfiguration()` at the bottom of all your code

ArrayField will now also automatically save and load your configuration

### Updating a Dropdown[​](https://docs.sirius.menu/rayfield/interaction/interactive-elements#updating-a-dropdown "Direct link to Updating a Dropdown")

You can update the dropdown's option list using `Dropdown:Refresh()`.

    Dropdown:Refresh({"New Option 1","New Option 2"}) -- The new list of options

Set the dropdown's currently selected option(s) using `Dropdown:Set()`. The option table can include multiple strings if you set MultipleOptions to true when creating the dropdown.

    Dropdown:Set({"Option 2"}) -- "Option 2" will now be selected
