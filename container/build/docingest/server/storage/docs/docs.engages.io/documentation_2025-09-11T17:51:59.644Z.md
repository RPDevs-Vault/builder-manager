# Server Settings | ​

![](https://docs.engages.io/~gitbook/image?url=https%3A%2F%2F876168839-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FpRFL8bwPdXCe6rKrnPK3%252Fuploads%252FtGLIkwa68DK4g4dsGRmm%252Fimage.png%3Falt%3Dmedia%26token%3Dc6b52a5d-aed0-4ebc-adcc-9f296f60139d&width=768&dpr=4&quality=100&sign=6288c234&sv=2)

**Under Point Settings -** allows you mainly to determine point settings in the server

*   **Reward Points**: Configure how many points users earn for their activities
    
*   **Daily Claim Rewards:** Set up daily reward points for users to claim
    
*   **Inflation Settings**: Every week the chosen percentage of points will be removed from all users' points
    

**Under Channel Settings** - allows you to set channels for sheet, winners, logs, and manage notify role.

*   **Sheet Channel** - Set a channel to receive automated Google Sheets for finalized events, raffles, or auctions
    
*   **Winner Channel** - Set the channel for announcing winners of various activities
    
*   **Logs Channel** - Set a New or Change the existing Log Channel
    
*   **Notify Role** - Determine the role that will receive notifications.
    

❗ **Under Wipe Leaderboard - to Reset the active leaderboard and turn every member points to 0**

# Engage Commands | ​

[](https://docs.engages.io/engage-discord/engage-commands#config-commands)

Config Commands


-----------------------------------------------------------------------------------------------

`Manage Messages Permission` gives your moderators access to use some of the commands.

Log channel receives the winners from auction or raffles.

*   `/config banner_reward`
    
    *   Gives Points To People Who Uses Your Banner
        
        *   `reward`
            
        *   `twitter_link`
            
        
    
*   `/config profile_reward`
    
    *   Give points to users who use your collection image.
        
        *   `reward`
            
        *   `contract_address`
            
        
    
*   `/config reaction`
    
    *   Enables reaction rewards, `Default: False`
        
    *   Users have to interact with `at least one of your tweets` before earning points from reactions
        
    
*   `/config claim`
    
    *   Allow certain role to claim points in a time interval.
        
    
*   `/config channel_reward`
    
    *   Set different points for different channels.
        
    
*   `/config winner_sheet`
    
    *   Set different channel to receive your winner list.
        
    
*   `**/config settings**`
    
    *   Change winner channel for rewards.
        
    *   Change engage logs channel.
        
    *   Change notify role on quest posts.
        
    *   Change rewards given per quest.
        
    *   Wipe user points.
        
    *   Enable message reward claims.
        
    *   Enable daily rewards for users.
        
    
*   `**/config server**`
    
    *   See your server information.
        
    *   Check engage boost statistics.
        
    *   Set socials for your server.

# Inflation Mode | ​

Engage now offers Inflation Mode in Servers

To activate this Server Mode:

1.  Run **/config settings** in engage-logs channels
    
2.  Choose **Point Settings**
    
3.  **Set your** Inflation Configuration
    

![](https://docs.engages.io/~gitbook/image?url=https%3A%2F%2F876168839-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FpRFL8bwPdXCe6rKrnPK3%252Fuploads%252FTFApy7IQiic3MR37FXh6%252Fimage.png%3Falt%3Dmedia%26token%3D22e5b3d4-14f4-4048-b5fe-9d50daf4d72b&width=768&dpr=4&quality=100&sign=67a4b637&sv=2)

### 

[](https://docs.engages.io/new-update-and-features/inflation-mode#how-it-works)

**How it works?**

*   Engage will ask you to determine PERCENTAGE of Inflation in the dedicated Server
    
*   Every week the chosen percentage of points will be removed from all users points
    
*   This will help servers to sustain their point generations and make users to spend their points accordingly

# Publish your Tweet | ​

![](https://docs.engages.io/~gitbook/image?url=https%3A%2F%2F876168839-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FpRFL8bwPdXCe6rKrnPK3%252Fuploads%252F2PUCx6OZXSgRNsv3WNST%252Fimage.png%3Falt%3Dmedia%26token%3Dee01056c-8443-42b9-a1ba-b0d6d0826252&width=768&dpr=4&quality=100&sign=b3b2d687&sv=2)

The Tweet Command provides three main input options.

*   **Tweet URL:** Input your tweet link here
    
*   **Type:** Choose your Interaction Type (choose one or combined Like / Retweet / Comment )
    
*   **Duration:** How long you want your members to be able to receive interaction Points
    

**Extra Options & Filtering**

![](https://docs.engages.io/~gitbook/image?url=https%3A%2F%2F876168839-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FpRFL8bwPdXCe6rKrnPK3%252Fuploads%252F8upb6OxSKtrEwGAtcZzF%252Fimage.png%3Falt%3Dmedia%26token%3D5e178623-79ac-44fa-bdce-6dcc912f4ffb&width=768&dpr=4&quality=100&sign=4da1c198&sv=2)

*   **Total Points:** Ensure members receive total points upon completing all interactions.
    
*   **Lead Engagers**: Extra Rewards for interacting in first 15minute
    
*   **Note:** Add Additional Notes for your Tweet
    
*   **Verified Only:** Only Allow Engage-Verified accounts to receive Points
    
*   **Assign-Role:** Members earn a specific role by engaging with this tweet.
    
*   **Minimum Character:** Determine a minimum character limit for comments
    
*   **Keyword Filter:** Enter a specific Keyword for Comments (example: #engage)
    
*   **Follow-Age:** Set a minimum follower and account age filter

--------------------------------------------------------------------------------------------------------------------------

To post a quest, use the following command:

* * *

[](https://docs.engages.io/engage-telegram/posting-quests#step-2-advanced-quest-posting-with-drafts)

**Step 2: Advanced Quest Posting with Drafts**


--------------------------------------------------------------------------------------------------------------------------------------------------------

If you want to include **extra options** like duration, follower requirements, or other advanced settings, follow these steps:

**1\. Open the Admin Panel**

*   The admin panel link was sent to the group admins when the bot was first set up.
    
*   Open the panel to manage your drafts and settings.
    

**2\. Create a Draft**

*   In the admin panel, you can create a new draft by:
    
    1.  Choosing a **draft name**.
        
    2.  Specifying settings like:
        
        *   Duration
            
        *   Follower requirement
            
        *   Account age
            
        *   Points or rewards
            
        
    3.  Saving the draft.
        
    

**3\. Post Using a Draft**

Once your draft is ready, you can post a quest using the following command:

Copy

    /tweet [tweet_url] [draft_name]

*   Replace `[tweet_url]` with the URL of the tweet.
    
*   Replace `[draft_name]` with the name of the draft you created.
    
*   The bot will automatically apply the settings saved in the draft.
    

![](https://docs.engages.io/~gitbook/image?url=https%3A%2F%2F876168839-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FpRFL8bwPdXCe6rKrnPK3%252Fuploads%252FnkS9KT3Xjt9GBDaXYaoR%252FScreenshot_4.png%3Falt%3Dmedia%26token%3Db5d546cd-8493-46e2-a40a-ba094d31bff3&width=768&dpr=4&quality=100&sign=72c56377&sv=2)

# Launch Giveaway | ​

[](https://docs.engages.io/engage-telegram/launch-giveaway#how-to-launch-a-giveaway)

**How to Launch a Giveaway**


----------------------------------------------------------------------------------------------------------------------

1.  **Navigate to the Giveaway Page** Open the **Launch Giveaway** page from the Admin Panel.
    
2.  **Fill Out Giveaway Details**
    
    *   **Raffle Title:** Enter the title of the giveaway. This will appear as the giveaway's name. _Example:_ "Exclusive NFT Raffle"
        
    *   **Raffle Cost:** Specify the number of points users need to spend to join the giveaway. _Example:_ 50 points
        
    *   **Raffle Spots:** Define the number of winners for the giveaway. _Example:_ 5 spots
        
    *   **Upload Image (Optional):** You can upload an image to make your giveaway more visually appealing.
        
    
3.  **Send Giveaway** Click the **Send Giveaway** button to publish the giveaway to your group.
    

* * *

#### 

[](https://docs.engages.io/engage-telegram/launch-giveaway#post-giveaways-faster-with-commands)

**Post Giveaways Faster with Commands**

If you prefer a faster method, you can use the `/raffle` command directly in your group:

Copy

    /raffle [Title] [Spots] [Cost]

*   **Title:** The name of the giveaway.
    
*   **Spots:** The number of available spots.
    
*   **Cost:** The points required to join. \[OPTIONAL\] Leave empty for costless raffle.
    

**Example:**

Copy

    /raffle Exclusive NFT Giveaway 100 50

This will launch a giveaway titled "Exclusive NFT Giveaway" with 100 spots, each requiring 50 points to join.
