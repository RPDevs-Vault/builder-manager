TapTools - API Documentation (1.8.1)
====================================

API Support: [support@taptools.io](mailto:support@taptools.io)
 URL: [https://taptools.io](https://taptools.io/)

TapTools is a data provider that offers onchain data for assets on the Cardano network.

[](https://openapi.taptools.io/#section/Introduction)
Introduction
------------------------------------------------------------------

This API includes many of the same endpoints that power the [TapTools website](https://taptools.io/)
. We have indexed onchain market data for both fungible and non-fungible tokens, as well as included endpoints to query raw onchain data.

[](https://openapi.taptools.io/#section/Rate-Limiting)
Rate Limiting
--------------------------------------------------------------------

This API implements rate limiting to ensure resources aren't overwhelmed. This rate limit varies by plan.

Collection assets
------------------------------------------------------------------------------------------------------

Get all NFTs from a collection with the ability to sort by price/rank and filter to specific traits. Categories and traits can be searched for by adding query params ex:(`policy=abc&hair=white,black&background=red`), you can have any combination of categories and traits ex:(`policy=abc&hair=blonde&background=blue`)

##### Authorizations:

_ApiAuthKey_

##### query Parameters

|     |     |
| --- | --- |
| policy<br><br>required | string<br><br>Example: policy=40fa2aa67258b4ce7b5782f74831d46a84c59a0ff0c28262fab21728<br><br>The policy ID for the collection. |
| sortBy | string<br><br>Example: sortBy=price<br><br>What should the results be sorted by. Options are `price` and `rank`. Default is `price`. |
| order | string<br><br>Example: order=asc<br><br>Which direction should the results be sorted. Options are `asc`, `desc`. Default is `asc` |
| search | string<br><br>Example: search=ClayNation3725<br><br>Search for a certain NFT's name, default is `null`. |
| onSale | string<br><br>Example: onSale=1<br><br>Return only nfts that are on sale Options are `0`, `1`. Default is `0`. |
| page | integer<br><br>Example: page=1<br><br>This endpoint supports pagination. Default page is `1`. |
| perPage | integer<br><br>Example: perPage=100<br><br>Specify how many items to return per page. Maximum is `100`, default is `100`. |

| numIntervals | integer<br><br>Example: numIntervals=180<br><br>The number of intervals to return, e.g. if you want 180 days of data in 1d intervals, then pass `180` here. |
