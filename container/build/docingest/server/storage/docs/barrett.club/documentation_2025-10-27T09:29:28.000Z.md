What's Barrett
---------------------------------------------------------

Barrett is a collection of blockchain trading data analyse tools that scrape and analyze all historical trading data on chain to obtain comprehensive views and alpha signals. With scanner, ai signals, trending info and other unique features, Barrett provides a large number of exclusive data indicators, which can better help select gems and identify scams.
