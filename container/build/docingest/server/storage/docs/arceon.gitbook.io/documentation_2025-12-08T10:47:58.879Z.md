### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#erode-blend-brush)

**Erode Blend Brush**

**Permission:** `arceon.brush.erodeblend`

`**/arcbrush erodeblend <size> [type]**`

Erodes, then blend balls terrain.

**Types:** Fill, Smooth, Lift, Melt and Clear.

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2FWDsoRdv.png&width=768&dpr=4&quality=100&sign=817f3b14&sv=2)

* * *

### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#cube-brush)

**Cube Brush**

**Permission:** `arceon.brush.cube`

`**/arcbrush cube <block> [size] [snapeAngle] [height] [width]**`

Creates cuboid shapes, you can use \[min\],\[max\] for each size parameter.

snapeAngle = angle intervals the cubes snap on (has to be a divisor of 360)

**Flags:**

**\-h flag** makes it hollow

**\-c flag** makes the height centered

**\-a flag** makes it angled based on your orientation

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2Fm0UJ7vg.png&width=768&dpr=4&quality=100&sign=54ba21ed&sv=2)

* * *

### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#spike-brush)

**Spike Brush**

**Permission:** `arceon.brush.spike`

`**/arcbrush spike <block> <length> <start> <end>**`

Uses convex selection.

**Start and end =** sphere size of start and end.

**Length =** Length of the spike.

**Flags:**

**\-n flag** adds noise to the spike shape.

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2FpqDwXI8.png&width=768&dpr=4&quality=100&sign=b003e3e0&sv=2)

* * *

### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#vine-brush)

**Vine Brush**

**Permission:** `arceon.brush.vine`

`**/arcbrush vine <block> [size] [chance] [min][,[max]]**`

**Chance =** place chance in percentage.

**Min and Max =** vine length.

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2FxauPaMp.png&width=768&dpr=4&quality=100&sign=e62c12ed&sv=2)

* * *

### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#crop-brush)

**Crop Brush**

**Permission:** `arceon.brush.crop`

`**/arcbrush crop <size> <min> [max]**`

Min and max data values between 0 and 8.

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2FH6KioDR.png&width=768&dpr=4&quality=100&sign=eac06be6&sv=2)

* * *

### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#revolve-brush)

**Revolve Brush**

**Permission:** `arceon.brush.revolve`

`**/arcbrush revolve <amount> [start] [end] [heightDiff]**`

> Uses cuboid selection.

**Start and end =** angle of start and end in degrees.

**HeightDiff =** Height difference between start and end.

**Flags:**

**\-p flag** makes the rotations regular polygon shapes

**\-r flag** makes the rotation reversed

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2FRTONfY0.png&width=768&dpr=4&quality=100&sign=d29a945b&sv=2)

* * *

### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/brushes#boulder-brush)

**Boulder Brush**

**Permission:** `arceon.brush.boulder`

`**/arcbrush boulder <block> [size] [amount] [height] [width]**`

you can use \[min\],\[max\] for each size parameter.

**Amount =** smoothness from 0 to 5 (0 is low poly)

**Flags:**

**\-h flag** makes the rocks hollow

**\-f flag** makes the rocks "fall" to the ground

**\-a flag** makes the rocks angled based on your orientation

![](https://arceon.gitbook.io/arceon-wiki/~gitbook/image?url=https%3A%2F%2Fi.imgur.com%2FzlV9EY4.png&width=768&dpr=4&quality=100&sign=b26b6524&sv=2)

Config


---------------------------------------------------------------------------------------

As of update 0.4.0, Arceon now offers a config located at: `/plugins/Arceon/config.yml`

If your config is missing any values, you may reference the [default config](https://arceon.gitbook.io/arceon-wiki/general-information/config#default-configurations)
 for your version.

[](https://arceon.gitbook.io/arceon-wiki/general-information/config#reload-command)

Reload Command

**Permission:** `arceon.admin`

`**/arceon reload**` - reloads the config.

Settings


-------------------------------------------------------------------------------------------

#### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/config#bstats-metrics)

bStats Metrics

Can the plugin collect anonymous usage metrics? Arceon metrics page: https://bstats.org/plugin/bukkit/Arceon/6481

Note: the server must be restarted if you change this value. Find global bStats settings in: `/plugins/bStats/config.yml`

Copy

    bstats-metrics: true

#### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/config#commands)

Commands

Command toggles. If there's a conflict with any of the built-in informational commands, disable them here. Note: the server must be restarted in order for updated command settings to take effect. All commands are enabled by default.

Copy

    commands:
      coffee: true
      suggest: true
      wiki: true

#### 

[](https://arceon.gitbook.io/arceon-wiki/general-information/config#selection-outline-particle)

Selection Outline Particle

# Arch-Tool | Arceon Wiki

**Permission:** `arceon.tool.arch`

`**//arch <block> <width> [height] [PillarSize] [Amount] [ArchHeight]**`

**PillarSize:** width of the pillars

**Amount:** amount of arches that have to be made

**ArchHeight:** % of the height

* * *

**Flags:**

*   **\-l flag** levels the height of the arches.
    
*   **\-d flag** stacks down the arches.
    
*   **\-p flag** straightens the path it follows.

# Dashes-Tool | Arceon Wiki

**Permission:** `arceon.tool.dashes`

`**//dashes <block> <DashLength> <GapLength> [width]**`

# Revolve-Tool | Arceon Wiki

**Permission:** `arceon.tool.revolve`

`**//revolve <amount> [start] [end] [heightDiff]**`

Uses cuboid selection.

**Start and end =** angle of start and end in degrees.

**HeightDiff =** Height difference between start and end.

**Flags:**

**\-p flag** makes the rotations regular polygon shapes

**\-r flag** makes the rotation reversed

**\-s flag** draws all points to the start of the revolve

# Terragen | Arceon Wiki

**Permission:** `arceon.tool.terragen`

`**//terragen [noiseType] [pattern] [height] [zoom] [wave]**`

Uses cuboid, sphere, cylinder and poly selections.

Use any slab or snow patterns for the system to use slabs or snow layers for more detailing.

`**//tgseed [seed]**`

Allows you to set the seed of the next Terragen, make sure to use the **\-s flag** or it won't work.

* * *

**Noise types:**

> **Simplex**, **Fractal**, **Billowy**, **Ridged**, **Turbulence**, **Electric**, **Euclidean**, **Natural** and **Poly**

**Zoom =** Scaling of the terrain, higher is more stretched out.

**Wave =** Adds a wavy effect to the terrain, default = 0.

* * *

**Flags:**

*   **\-s flag** keep the seed the same as before.
    
*   **\-v flag** adds a vignette mask over the terragen which smooths out the edges.
    
*   **\-e flag** layers it on top of existing terrain.
    

> **Euclidean**, **Manhattan** and **Natural**

*   **\-r flag** randomizes the cellular noises.
    

> **Fractal**, **Billowy**, **Ridged**, **Turbulence** and **Electric**

*   **\-n flag** adds more noise.
