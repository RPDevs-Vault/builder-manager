> Bộ tài liệu sau sẽ hướng dẫn bạn dùng **n8n** và gói **n8n-nodes-ultimate** để tự động hóa Zalo, Facebook và tải video mạng xã hội. Bao gồm: `Zalo User`, `Zalo Trigger`, `Zalo Bot`, `Facebook User`, `Download Social Videos`, `Message Debounce`.

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate#diem-noi-bat)

Điểm nổi bật


-----------------------------------------------------------------------------------------

**Trigger**

*   **Zalo User**: tin nhắn, lời mời kết bạn, thả tim của tài khoản cá nhân.
    
*   **Zalo Bot**: nhận tin nhắn đến của khách hàng.
    
*   **Facebook User**: tin nhắn group messenger và nghe thông báo của tài khoản facebook.
    

**Action**

*   **Zalo User**: gửi tin nhắn, quản lý bạn bè/nhóm, sticker, poll, tag, catalog
    
*   **Zalo Bot**: API chính thức để gửi tin và webhook
    
*   **Facebook User**: nhắn tin, quản lý bạn bè, nhóm chat, đăng bài (unofficial)
    
*   **Download Social Videos**: tải video từ Url/Kênh TikTok/YouTube/Facebook Reels
    
*   **Message Debounce**: gộp tin nhắn theo thời gian
    

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate#luu-y-quan-trong)

Lưu ý quan trọng


-------------------------------------------------------------------------------------------------

*   Zalo User và Facebook User là **Unofficial**, hãy cẩn trọng với tài khoản thật.
    
*   Zalo Bot sử dụng **Official** API.
    
*   Cần cài thêm `python3` và `ffmpeg` khi tải video.
    
*   Khuyến nghị n8n v0.187 trở lên để tương thích tốt.
    

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate#yeu-cau)

Yêu cầu


-------------------------------------------------------------------------------

*   n8n v0.187 hoặc mới hơn
    

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate#ho-tro)

Cài qua Community Nodes (khuyến nghị)

1.  Mở n8n Editor.

3.  Chọn **Install** và nhập `n8n-nodes-ultimate`.

    cd YOUR_N8N_DIRECTORY
    npm install n8n-nodes-ultimate

###

Cài qua Docker

Copy

    FROM n8nio/n8n
    
    USER root
    RUN cd /home/node/.n8n/custom/node_modules && \
        npm install n8n-nodes-ultimate
    USER node

Parameters


------------------------------------------------------------------------------------------------------

*   **Name Account**: tên gợi nhớ cho tài khoản Zalo.
    
*   **User ID**: ID người dùng Zalo.
    
*   **Phone Number**: số điện thoại Zalo.
    
*   **Proxy**: proxy HTTP dùng cho các request Zalo.
    
*   **Telegram Bot Token**: token bot Telegram để nhận thông báo (nếu dùng).
    
*   **Telegram Chat ID**: chat\_id Telegram để nhận thông báo.
    
*   **Cookie**: cookie từ phiên đăng nhập Zalo.
    
*   **IMEI**: mã IMEI từ phiên đăng nhập Zalo.
    
*   **User Agent**: user agent từ phiên đăng nhập Zalo.
    
*   **Support Code**: mã hỗ trợ cho Zalo API.
    
*   **License Key**: license key cho Zalo API.

Lưu ý

*   Credentials này thường được tạo tự động từ Zalo Login By QR.
    
*   Nếu cần, bạn vẫn có thể tạo credential thủ công để dùng với “Select a custom credentials ID”.

*   Cách lấy cookies user: mở https://web.facebook.com → dùng J2TEAM Cookies → Export as text → dán JSON vào trường Cookies User.
    
*   Cách lấy cookies bussiness: mở https://business.facebook.com → → →

###

Credentials account

*   **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo
    

### 

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate/danh-sach-nut/zalo/trigger#trigger-on-gom-cac-su-kien)

Trigger On gồm các sự kiện

*   User Messages: _tin nhắn chat riêng_
    
*   Group Messages: _tin nhắn nhóm chat_
    
*   Group Event: _một vài sự kiện xảy ra trong nhóm_
    
*   Friend Event: _sự kiện kết bạn, lời mời_
    
*   Undo: _ai đó thu hồi tin nhắn_
    
*   Typing: _ai đó đang soạn tin_
    
*   Reaction: _ai đó thả cảm xúc_
    
*   Seen Message: _ai đó đã xem tin nhắn_
    

### 

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate/danh-sach-nut/zalo/trigger#parameters-message-event)

Parameters Message Event

*   **Media Debounce**: Gộp nhiều ảnh/video gửi cùng lúc thành 1 sự kiện
    
    *   Khi ai đó gửi 10 ảnh cho bạn 1 lần, thay vì nhận 10 sự kiện nó sẽ gôm lại thành 1
        
    *   Bổ sung trường `href_photo`, `href_video` trong `{ content }`
        
    *   Chờ tối đa 10s, không thể Execute step (mạng yếu, video nặng sẽ bỏ qua và thành sự kiện lẻ)
        
    
*   **Use Filtered Output**: Chuẩn hóa kết quả để dễ dùng
    
    *   Thêm trường `{ filter }` đã lọc dữ liệu text, href, href\_photo, href\_video.
        
    *   `chat.photo`: lọc url, title
        
    *   `chat.video`: lọc url, title
        
    *   `chat.recommended`: danh thiếp: phone, qrCodeUrl, uid, thumb
        
    *   `chat.location`: vị trí: description, latitude, longitude, href
        
    
*   **Filtering Thread IDs**: Lọc theo thread ID (không thể Execute step)
    
    *   **Allowed Thread IDs**: Chỉ nhận tin nhắn từ các thread ID (phân cách bằng dấu phẩy). Để trống = lắng nghe tất cả
        
    *   **Blocked Thread IDs**: Bỏ qua tin nhắn từ các thread ID này (phân cách bằng dấu phẩy)
        
    
*   **Filtering Keyword**: Lọc `message` theo từ khóa (không thể Execute step)
    
    *   **Listen Only Keywords**: Chỉ nhận tin nhắn chứa ít nhất một trong các từ khóa này (phân cách bằng dấu phẩy, ít nhất 2 kí tự)
        
    *   **Ignore Keywords**: Bỏ qua tin nhắn chứa bất kỳ từ khóa nào trong danh sách này (phân cách bằng dấu phẩy, ít nhất 2 kí tự)
        
    
*   **Continue On Login Fail**: không làm dừng workflow nếu có 1 trigger login fail (báo qua telegram)
    
    *   **Telegram Bot Token**: Bot token (để trống nếu credential đã có)
        
    *   **Telegram Chat ID**: ID nhóm chat (để trống nếu credential đã có)
        
    
*   **Self Listen (bắt buộc)**: Nhận sự kiện do chính bạn thực hiện
    
    *   **Self Messages Only**: chỉ nhận sự kiện User/Group Message (không thể Execute step)
        
    
*   **Auto Restart (Hours)**: Tự động khởi động lại kết nối sau mỗi x giờ giúp ổn định (0=tắt, không khởi động lại workflow/n8n)

*   **Mode**: Zalo dùng key mặc định cho trigger Zalo; Custom cho phép chỉ định key/content từ JSON.
    
*   **Wait Time (Seconds)**: thời gian chờ gom tin nhắn tiếp theo (tối đa 60s).
    
*   **Max Batched Messages**: số tin nhắn gộp tối đa (tối đa 10).
    
*   **Separator**: ký tự nối các tin nhắn (mặc định là xuống dòng).
    

### 

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate/danh-sach-nut/message-debounce#zalo-mode)

Zalo Mode

*   **Ignore IsSelf**: bỏ qua tin nhắn của chính mình (tắt có thể gây vòng lặp).
    
*   **Extract Text From Photo/Video**: gom text trong ảnh/video (nếu có) và đặt `data.content.title` = "".
    
*   **Content Fields**: đường dẫn tới nội dung cần gom, mặc định `data.content`.
    

### 

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate/danh-sach-nut/message-debounce#custom-mode)

Custom Mode

*   **Key Fields**: danh sách field tạo key duy nhất, ví dụ `id,userId,type`.
    
*   **Content Fields**: đường dẫn tới nội dung cần gom, ví dụ `data.content` hoặc `payload.text`.
    
*   **Drop When Path is true (boolean)**: nếu path trả về `true` thì bỏ qua item (ví dụ `isSelf`).
    

### 

[hashtag](https://codedao12.gitbook.io/n8n-nodes-ultimate/danh-sach-nut/message-debounce#luu-y)

Lưu ý

*   Node sẽ reset thời gian chờ nếu có tin mới và chỉ trả output khi hết thời gian chờ.
    
*   Ví dụ tạo key: `id,data.uid` và lấy nội dung `data.text` từ JSON.
    
*   Dùng được cho mọi nền tảng có trigger (không chỉ Zalo/Facebook).
