# 通用動作 | TJService                                                                                             Less Cost & More Power

Delay[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#delay)

_延遲執行動作_

Copy

    action:
      +: TJAction
      delay_1:
        message: 'wtf' # 此訊息將在 10 秒後才發送
        delay: '10s'

Group[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#group)

_打包動作成為動作組，建議配合其他動作使用。_

Copy

    action:
      +: TJAction
      random_1:
        message_1: '&c你沒抽中 :('
        group_1:
          message_1: '&a恭喜獲得 &b鑽石 &fx10&a!'
          console_1: 'give %player_name% diamond 10'

If[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#if)

_如果?_

Copy

    action:
      +: TJAction
      if_1:
        condition:
          hasperm_1: 'hello.world'
        n: 
          message: '&c你沒有 hello.world 權限'
        y: 
          message: 'Hello World!'

Copy

    action:
      +: TJAction
      if_1:
        expr: '%player_y% > 63 and %player_level% >= 10' # 簡易運算式
        y:
          message: '&b你在海平面上且你有 10 等!'
        n:
          message: '&c你在海平面下或未達 10 等!'

Loop[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#loop)

_來點重複的樂趣?_

Copy

    action:
      +: TJAction
      loop_1:
        time: '3' # 此參數接受運算式，故 '' 不可缺少。
        message: '&e我會跑三次。'

Random[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#random)

_多選N?_

Copy

    action:
      +: TJAction
      random_1:
        amount: 1 # (選出結果的數量，選填)
        message_1: '&a我是一號。'
        message_2: '&e我是二號。'
        message_3: '&c我是三號。'

Set[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#set)

_設定參數為運算式結果(請見 If)。_

**變數僅存在記憶體，重啟會消失。**

Copy

    action:
      +: TJAction
      set_1:
        id: 'hello'
        expr: '%player_y% > 63'
        keep: false # 是否只在不存在時設定，(選填)
      if_1:
        expr: '{hello}' # 使用 "{變數名}" 存取自訂變數
        y:
          message: '&b你在海平面上!'
        n:
          message: '&c你不在海平面上!'

HasPerm[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#hasperm)

權限檢查，回傳是否有權限

Copy

    action:
      +: TJAction
      if_1:
        condition:
          hasperm_1: 'hello.world'
        n: 
          message: '&c你沒有 hello.world 權限'
        y: 
          message: 'Hello World!'

And[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#and)

與運算

Copy

    action:
      +: TJAction
      if_1:
        condition:
          and_1:
            hasperm_1: 'hello.world'
            hasperm_2: 'goodbye.world'
        y: 
          message: 'Hello World!'
          message_1: 'Goodbye World!'
        n: 
          message: '&c你沒有 hello.world + goodbye.world 權限'

Or[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#or)

或運算

Copy

    action:
      +: TJAction
      if_1:
        condition:
          or_1:
            hasperm_1: 'hello.world'
            hasperm_2: 'hello_world'
        y: 
          message: 'Hello World!'
        n: 
          message: '&c你沒有 hello.world 或 hello_world 權限'

Check[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#check)

多重檢查

Copy

    action:
      +: TJAction
      check:
        conditions:
        - condition: 3 > 1
          success: '好耶，3 > 1'
          fail: '&c這不太可能發生@@?'
        - condition:
            not: '%player_health% > 10'
          fail: '你血量太多了...'
        success: # 所有檢查通過
          message: '&aOk'
          console: 'give %this% golden_apple'
        fail: # 任一檢查失敗，可不寫
          message: '&cFail'

Broadcast[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#broadcast)

將動作委託給所有玩家

Copy

    action:
      +: TJAction
      broadcast:
        message: '現在，大家都知道我很吵了 ><'

AsConsole[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/tong-yong-dong-zuo#asconsole)

將動作委託給後台

Copy

    action:
      +: TJAction
      asconsole:
        message: '後台看到我了??'

----------------------------------------------------------------------------------------------------------------------------------

但倘若你不接受我們的建議，我們並不會免費協助修改你未遵循委託須知的部份。

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#wei-tuo-wen-yao-su)

委託文要素


--------------------------------------------------------------------------------------------

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#yi-xia-suo-shu-ru-wei-ming-zhu-bu-de-you-zheng-yi-fu-wu-tiao-kuan-nei-you-ti-ji)

\# 以下所述如未明注，不得有爭議（服務條款內有提及）

*   版本 (**如未明注，則會自動採用當前時期最多人委託的版本編寫**)
    
*   依賴插件
    
*   設定檔案架構 (如未明注，則自由發揮)
    
*   委託內容
    
*   指令 (**如需特殊離線玩家、跨分流玩家支援請標註，否則因架構一律不支援**)
    
*   PlaceholderAPI 變數
    
*   資料庫儲存需求 (**MySQL 儲存不代表可跨分流**)
    
*   **插件可否跨分流**
    

*   **這段話有些直接但** 如果可以，請順帶打上你的預算，方便我 (如果100塊想委託完整的插件那可以先左轉了，不好意思) 開伺服器本來就該準備資金，Cheap Fast Good 平衡法則， 不會有又便宜、又快、又好的委託服務，因為這種服務只會被濫用 TJService 已經盡力維持平衡，提供 Cheap Fast Good 的委託服務直接拒絕
    

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#wei-tuo-zhuan-an-bao-jia-you-xia-lie-duo-zhong-yin-su-jue-ding)

委託專案報價由下列多種因素決定


--------------------------------------------------------------------------------------------------------------------------------------------------

*   需要自訂程度
    
    *   有多語言需求
        
    *   **非玩家可視內容翻譯需求**
        
        委託時預設會免費提供近乎所有玩家看的到GUI標題、物品、訊息的修改 但如果是管理員的選單、內容等需要外拉翻譯，則會多收費
        
    
*   委託文表達清晰度
    
*   當前忙碌度
    
*   委託內部代碼(Spigot/Paper API等外)觸及度
    
*   版本跨度(最新版或太舊的版本皆會較貴)
    
*   API支援度(目前尚未有 Velocity API)
    
*   封包使用度
    
*   態度(凹價格、態度不佳只會搞的雙方心情不爽)
    
*   ... 等等

你可以做的，即是改善委託文表達清晰度


-----------------------------------------------------------------------------------------------------------------------------------------------------

**我相信沒有人喜歡看文言文，我們也一樣。**

良好的表達能力，是現代生活不可或缺的課題，對於委託亦是如此。

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#xia-lie-shi-yi-xie-ni-ke-yi-zhu-yi-de)

下列是一些你可以注意的


---------------------------------------------------------------------------------------------------------------------

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#wei-tuo-wen-wan-zheng-xing)

委託文完整性

*   我們需要一篇完整的文章，所以請不要對話的的格式委託(尤其在 Discord 頻道)， 倘若因訊息未明確界定範圍，導致多做或少做了某些部份，請自行負責。
    
*   請整理成一篇，我們僅會看一最後整理過的委託文，如有爭議則自行負責
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#wei-tuo-wen-zhi-jie-xing)

委託文直接性

*   我們不需要知道你的用途
    
*   我們只需要知道你要什麼功能，先做什麼後做什麼
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#wen-zhang-shun-xu)

文章順序

*   假定你設定了 A、B、C 三個系統
    
*   而其中，A需要用到B的內容、B需要用到C的內容
    
*   委託文順序**請依 C、B、A 從上到下打**，才能確保內容連貫性
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#gui-jie-mian)

GUI介面

*   建議使用箱子排，並以文字補述個別按鈕內容
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#config-she-ding-dang-ge-shi)

Config設定檔格式

*   如果你有能力，可以直接將期望的 yml 打出來
    
*   但只是讓我們更好設計，並不會 100% 相符合
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#wen-zhang-ge-shi)

文章格式

*   對於簡單的插件，可以使用簡易的文字表達
    
*   但當簡單的插件變為複雜的系統， 我們會建議使用 **PDF** 或者是 **Markdown** 等格式語言來明確表達你的需求
    
*   如有需要，[自行架設的筆記系統](https://note.tjservice.org/)
    可供你使用，以 **Markdown** 格式來撰寫委託文

以下一個項目為單一則聊天訊息(委託文完整性、直接性等 不良示範)

*   類似mm的RandomSpawn
    
*   支援mm
    
*   不過他的生成方式不同
    
*   他能在設定時間內 重複在設定世界 任意地方(能設定min max範圍) 生成 (已編輯)
    
*   並且生成時會發出公告
    

Copy

    message:
          '%spawn_mob%在 %spawn_world% 座標 X:%spawn_x% Y:%spawn_y% Z:%spawn_z% 的地方生成了'

*   跟MythicAnn一樣
    
*   每隻怪物單獨設定
    
*   還能夠設定PositionType: \[LAND/SEA\]
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/wei-tuo-xu-zhi#jian-yi-xiu-gai-ban-ben)

建議修改版本

> 隨機生成 MM 怪物

Copy

    SkeletalKnight:
      world: 
      - world
      range: "-500,-64,-500 ~ 500,320,500"

# TJAction | TJService                                                                                             Less Cost & More Power

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction#tjaction-shi-shen-mo)

TJAction 是什麼?


---------------------------------------------------------------------------------------------------------

TJAction 是由 TJService 開發的 YAML 動作系統，使用他可以觸發一些簡單動作。 比如傳送訊息同時撥放聲音、執行指令、關閉選單等。

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction#ji-ben-ge-shi)

基本格式


-----------------------------------------------------------------------------------------

Copy

    action:
      +: TJAction
      playsound_1: ENTITY_PLAYER_LEVELUP
      actionbar_1: '%message%'
      title_1:
        fadeOut: 10
        fadeIn: 10
        title: '%message%'
        stay: 100

在 YAML 的 TJAction 物件下，每一個 key 都是一個動作。 部分動作有簡寫(一行字串構成)，也可以是資料結構動作。

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction#dong-zuo-de-zhen-ce-qu-jue-wu-key-de-kai-tou)

動作的偵測取決於 key 的開頭


------------------------------------------------------------------------------------------------------------------------------------

比如說 `title`, `title1`, `title_wtf`, `title+`，都會被視為動作 title。

# 簡易運算式 | TJService                                                                                             Less Cost & More Power

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/jian-yi-yun-suan-shi#zai-tjaction-zhong-ni-ke-yi-bian-xie-yi-xie-jian-yi-de-yun-suan-shi-lai-bu-zhu-ji-suan-gong-cha-jian)

在 TJAction 中，你可以編寫一些簡易的運算式來補助計算、供插件本身功能取值。

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/jian-yi-yun-suan-shi#fu-hao-ding-yi)

符號定義

*   基礎運算 + - \* / % (取餘運算)
    
*   資料比對 =, !=, >, <, >=, <=
    
*   變數設定 a := 100
    
*   次方 10\*\*100
    
*   階乘 10!
    
*   邏輯運算 a and b, a or b, !a
    
*   布林 true/false
    
*   字串 'hello world'
    
*   數字 1234
    
*   函數呼叫 func(1,2,3)
    
*   角度 30deg
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/jian-yi-yun-suan-shi#te-shu-ding-yi)

特殊定義

*   字串可乘以數字 'hello' \* 3 ⇒ 'hellohellohello'
    
*   字串 % 字串 ⇒ 忽略大小寫比對 'hello' % 'Hello' ⇒ true
    
*   字串 > 字串 ⇒ 左邊是否以右邊開頭 'hello world' > 'hello' ⇒ true
    
*   字串 < 字串 ⇒ 左邊是否以右邊結尾 'hello world' < 'world' ⇒ true
    

### 

[](https://docs.tjservice.org/cha-jian-wei-tuo/she-ding/tjaction/jian-yi-yun-suan-shi#ji-ben-han-shu)

基本函數

*   not(bool) 取反 not(1=1) ⇒ false
    
*   cos(rad) 餘弦 cos(90deg) ⇒ 0
    
*   sin(rad) 正弦 sin(90deg) ⇒ 1
    
*   rad(deg) 弧度轉弳度 rad(30) ⇒ 0.5235987756 (等同 30deg)
    
*   deg(rad) 弳度轉弧度 deg(0.5235987756) ⇒ 30.0000000001 (多的是誤差)
    
*   abs(number) 取絕對值 abs(-1.5) ⇒ 1.5
    
*   sum(a, b, ...) 將所有數值加總 sum(1, 2, 3.5) ⇒ 6.5
    
*   avg(a, b, ...) 平均值(若沒有參數，結果為 0) avg(2, 4, 6) ⇒ 4
    
*   max(a, b, ...) 找出最大值 max(1, 5, 3) ⇒ 5
    
*   min(a, b, ...) 找出最小值 min(1, 5, 3) ⇒ 1
    
*   clamp(x, min, max) 限制 x 的值不超出 \[min, max\] 範圍 clamp(10, 0, 5) ⇒ 5
    
*   sqrt(number) 平方根 sqrt(9) ⇒ 3
    
*   floor(number) 無條件捨去（向下取整） floor(2.9) ⇒ 2
    
*   ceil(number) 無條件進位（向上取整） ceil(2.1) ⇒ 3
    
*   round(number) 四捨五入 round(2.5) ⇒ 3
    
*   random() 隨機布林值 random() ⇒ true/false
    
*   random(probability) 以指定機率返回 true，否則 false random(0.25) ⇒ true/false (25% 機率為 true)
    
*   random(a, b) 回傳 a 到 b 間的隨機實數 random(1, 3) ⇒ 2.184
    
*   randomint(a, b) 回傳 a 到 b 間的隨機整數(含兩端) randomint(1, 3) ⇒ 2

修改插件

在這個千變萬化的世界中，每個人的想法不同，自然會產生各式各樣的代碼及寫法(Code Style)。

修改插件必須理解別人的邏輯。

不使用API

> *   進入排隊期間 - 請在這段時間內完成付款(如果有很多人可以先付款排隊搶位)
>     
> *   開始製作 - 如果是使用文字訊息，期間會使用**打勾 或 刪除的方式回報進度**
>     
> *   \[如有**尾款**\] 提供**預覽查看**後等待收到尾款
>     
> *   **送出專案檔案**
>     
> *   ❌ 拒絕 可能原因為無能力製作、**委託文不標準** 或 **缺乏必要資訊**，一般來說補上資訊即可
>

\# 四/委託福利


------------------------------------------------------------------------------

1.  可要求原代碼，**不須付費**，原代碼亦不保證能編譯
    
2.  \[插件類委託\] 會使用自製API 加速編寫 並也**以API的方式降低價格**
    
3.  \[插件類委託\] **玩家可視部分**，設定**不依照設定數量多付錢**， 且基本上能設定的都會給設定(訊息、資料值、物品顯示..)
