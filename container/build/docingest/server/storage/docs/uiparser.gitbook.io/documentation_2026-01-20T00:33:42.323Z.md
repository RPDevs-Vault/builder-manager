# Frames | UIParser Docs

Copy

    BeginChild( frameName, frameSize, flags );
    {
        // content

    EndChild( );

# Tabs list | UIParser Docs

A list of tabs must be named **"tabs"** and it expands in

Copy

    widgets->nav.drawtabs( );

**EXAMPLE OF DRAWTABS CODE**

Copy

    void c_widgets::widget_nav::drawtabs( )
    {
        SetCursorPos( vec2{ 14, 44 } );
        BeginChild( "tabs", vec2{ 168, 0 }, 96, 128 );
        {
            // "tabs" frame content
        }
        EndChild( );
        
        tab_anim = ImLerp( tab_anim, tab_animdest, GetIO( ).DeltaTime * 17 );
        subtab_anim = ImLerp( subtab_anim, subtab_animdest, GetIO( ).DeltaTime * 17 );
        
        if ( tab_anim < 0.05f ) {
            tab_animdest = 1.f;
            current = next;
        }
        
        if ( subtab_anim < 0.05f ) {
            subtab_animdest = 1.f;
            tabs[current].current = tabs[current].next;
        }
    }

# Subtabs list | UIParser Docs

A list of subtabs must be named **"subtabs"** and have this structure

Copy

    // Frame "subtabs":
    // -- Frame "list":
    // ---- Subtabs

It expands in

Copy

    widgets->nav.drawsubtabs( );

**EXAMPLE OF DRAWTABS CODE**

Copy

    void c_widgets::widget_nav::drawsubtabs( )
    {
        if ( tabs[current].subtabs.empty( ) )
        return;
            
        PushStyleVar( ImGuiStyleVar_Alpha, GImGui->Style.Alpha * tab_anim );
        SetCursorPos( vec2{ 14, 14 } );
        BeginGroup( );
        {
            
            // "list" frame content
        }
        EndGroup( );
        
        PopStyleVar( );
    }

# Styles | UIParser Docs

To ensure that a color configuration is registered in the table, a dedicated **Rectangle** element may be created and filled with the required color. This element must be named using the `pcol_*` naming convention. UIParser detects such elements and automatically extracts the color value for inclusion in the configuration table.
