# Getting Started | Plague API

[](https://docs.plaguecheat.cc/getting-started#directory)

Directory


------------------------------------------------------------------------

*   All related files (configs, scripts, models, etc) need to be placed in proper folders
    
*   Scripts & config files go under c:\\plaguecheat.cc
    
*   Models go under the Counter-Strike 2 root folder, then into game/csgo/characters
    

For documentation on FFI, refer to [LuaJIT's FFI tutorial](https://luajit.org/ext_ffi_tutorial.html)
