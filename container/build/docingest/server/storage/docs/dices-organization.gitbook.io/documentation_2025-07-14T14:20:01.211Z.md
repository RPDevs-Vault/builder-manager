# $ROLL utilities | Farmroll

1\. Campaign Launch Programs

Description: Businesses and users use $ROLL to pay for creating questing campaigns and unlocking advanced features.

Details: Projects and community members can create engagement campaigns with $ROLL:

*   Project-Created Campaigns (Tier 1): Businesses and projects use $ROLL to create questing campaigns and customize engagement pathways. They can also pay $ROLL to unlock premium features, such as campaign analytics, performance metrics, and automation options.
    
*   User-Created Campaigns (Tier 2): Community members can create their own challenges and reward participation using $ROLL, fostering interaction between users.
    

Additionally, $ROLL can be used to unlock analytics dashboards and premium customization options for deeper user engagement.

* * *

2\. Upgrade Mechanics

Description: Users and projects burn $ROLL tokens to unlock new features and upgrade AI agents or campaign functionality.

Details:

*   AI Agent Upgrades: Users burn $ROLL to unlock new features for AI agents, such as image generation, Discord integration, voice-based interactions, and on-chain capabilities.
    
*   Questing System Upgrades: $ROLL can also be burned to unlock premium questing features like detailed analytics, automation, and loyalty point tracking.
    
*   Reward Boosters: Users can purchase reward boosters with $ROLL to temporarily increase APY or campaign rewards during special events, adding a dynamic, competitive element and enhancing $ROLL's utility.
    

Each upgrade permanently removes $ROLL from circulation, contributing to the deflationary model while enhancing user experiences and campaign reach.

* * *

3\. AI Agents and Liquidity

Description: Users can launch either a non-coin AI agent that requires burning $ROLL to unlock features or a coin-backed AI agent that requires $ROLL liquidity.

Details:

*   Non-Coin AI Agents: Launching a non-coin AI agent allows users to automate engagement tasks without liquidity requirements. $ROLL is burned to unlock premium features, such as image, voice and video generation, Discord, Telegram, Tik-tok, and Instagram integrations, and on-chain activities.
    
*   Coin-Backed AI Agents: These agents require a specific amount of $ROLL as liquidity, which remains locked to maintain token stability. These agents generate revenue, with a portion shared with $ROLL stakers.
