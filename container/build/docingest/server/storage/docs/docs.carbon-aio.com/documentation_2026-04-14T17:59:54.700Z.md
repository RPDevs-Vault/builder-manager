# Auteuil | Carbon

circle-info

To **activate** **ACO** mode with credit card, you need to fill the `profiles.csv` file. These are the columns that you need to fill : `ProfileName` , `CcNumber` , `CcExpiryMonth` , `CcExpiryYear` , `CcCVV`. Make sure to check[profiles.csv](https://docs.carbon-aio.com/config-csv-format/profiles.csv)
 to avoid any issues. Leave these columns empty if you want to checkout with giftcards.

### 

[hashtag](https://docs.carbon-aio.com/sport/auteuil#tasks.csv)

Tasks.csv

*   `**Mode**` **(required)** : The mode used by the task `NORMAL` or `IGNORE` .
    

NORMAL QUEUE

IGNORE

`NORMAL QUEUE` mode will handle the queue itself.

`IGNORE` mode will ignore the task and won't start it.

* * *

*   `**Mail**` **(required)****:** The mail of your PSG account.
    

triangle-exclamation

Your PSG account needs to have access to Auteuil offer, otherwise it won't be able to see the `CATEGORIE 18`

The category 18 is only specific to Auteuil.

* * *

*   `**Password**` **(required)** **:** The password of your PSG account.
    

* * *

*   `**EventName**` **(required)** **:** This will be for you to identify which task is running.
    

> (eg : `PSG vs LENS`)

* * *

*   `**EventLink**` **(required)** : You can only put one type of link in this column.
    

1.  The link of Ticketplace :
    

circle-check

[`https://billetterie.psg.fr/fr/second/match-foot-masculin-paris-vs-lens-6/`arrow-up-right](https://billetterie.psg.fr/fr/second/match-foot-masculin-paris-vs-lens-6/)

* * *

*   `**Delay**` **(required)** : It's the time duration in milliseconds a task will wait before each attempt to check stock. Must be equal or above `0` (eg : `2000`)
    

* * *

*   `**QuantitiesFilter**` **(required)** : The different quantities you want to target on a task.
    

> (eg : `1;2;3;4` Here the bot will only target solo, duo, trio and quartor on the task) (eg : `2;3` Here the bot will only target duo and trio on the task) (eg : `2` Here the bot will only target duo on the task)

* * *

* * *

*   `**MinPriceFilter**` **(optional)** **:** The minimum price you want to target for Auteuil listing.
    

> (eg : `100` Here the bot will focus all the Auteuil listing with price above 100 euros)

* * *

*   `**MaxPriceFilter**` **(optional)** : The maximum price you want to target for Auteuil listing.

circle-info

Workers define how many captcha-solving processes run in parallel on your machine.

*   **1 worker** = uses roughly **1 CPU core**
    
*   **2 workers** = uses roughly **2 CPU cores**
    
*   **12 workers** = uses roughly **12 CPU cores**
    

The more workers you use:

*   ✅ Faster captcha solving (more tasks processed at the same time)
    
*   ❌ Higher CPU usage and overall system consumption
    

For now, We recommend using **2–4 workers** as a good balance between performance and resource usage.
