# What is a transaction hash? – Uniswap Labs

What is a transaction hash?
===========================

A blockchain transaction hash (also called a transaction ID) is a unique set of characters assigned to every transaction.

Blockchain explorers like [etherscan.io](https://etherscan.io/)
 can be used to review a transaction and find the transaction hash. The transaction hash is best sent by copying/pasting from your wallet or a block explorer. It's recommended to double check that you've copied the hash correctly.

It is not recommended to send screenshots of your transaction data, as they are not always clear enough to read and process the information. Sending screenshots may delay a support agent's resolution to your issue.

# How to send tokens using the Uniswap Wallet – Uniswap Labs

How to send tokens using the Uniswap Wallet
===========================================

Send tokens to another address in the Uniswap Wallet by selecting the recipient, choosing a supported token, entering an amount, and reviewing the transfer before sending. The guide also highlights important checks to ensure the address, network, and token are supported.

You can send tokens on [any supported network](https://support.uniswap.org/hc/en-us/articles/14569415293325-Networks-on-Uniswap)
 to another address using the Uniswap Wallet.

**All transfers on the blockchain are permanent and cannot be reversed. Therefore, it is important you make sure of the following before you proceed with sending tokens:**

*   Your wallet supports the token you are trying to send
*   Your wallet, address, or dapp supports the network you are sending the token on
*   The wallet address you are sending to is correct

**How to send tokens using the Uniswap Wallet:**

1.  Open your Uniswap Wallet and select the send icon.  
     
2.  Enter the address you would like to send to the tokens to.  
      
    You can enter the receiving address in four ways:
    1.  Enter the Uniswap Username of the receiving address (i.e. callil).
    2.  Enter the ENS of the receiving address. (i.e. callil.eth).
    3.  Paste the address from your clipboard. (i.e. 0x4defA30195094963cFAc7285d8d6E6E523c7f90D).
    4.  Scan the receiving address's QR code by selecting the scan icon in the search bar.  
         
3.  Select the token menu.   
      
    ![Frame 1321321211.png](https://support.uniswap.org/hc/article_attachments/31729572031117)  
     
4.  Search for and select the token you would like to send.   
      
    You can search for a token in three ways:
    1.  Enter the contract address of the token (i.e. 0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48).
    2.  Enter the name of the token. (i.e. USDC).
    3.  Scroll through the list of tokens in your wallet.  
         
5.  Enter the token amount you would like to send, or select the “Max” option.   
    The “Max” option automatically enters the total amount of the token you have available in your wallet.  
     
6.  Select “Review transfer”.   
      
    This is your final opportunity to make sure the address you are sending to is correct and is supported by the network you using to send tokens you are sending funds over.  
      
    ![Frame 1321321210.png](https://support.uniswap.org/hc/article_attachments/31729572038797)  
     
7.  Select “Send”.  
      
     
8.  Congratulations, you have now sent tokens to another address!  
      
    Once the send is successful you can select “View transaction” to see the send on a blockchain explorer or select “Close”.  
      
    ![Frame 1321321209 (1).png](https://support.uniswap.org/hc/article_attachments/31729591167885)

# I bought crypto using the Uniswap Wallet, but I have not received it yet. – Uniswap Labs

I bought crypto using the Uniswap Wallet, but I have not received it yet.
=========================================================================

If you bought crypto and have not received it, the transaction may still be processing.

Sometimes, it can take more time than expected to complete the transaction. If you bought crypto, it should appear after 30 seconds of the purchase being completed. However, it may take up to 5 minutes to appear in your wallet.

**To check your transaction's status on the blockchain:**

1.  Select "Activity" to view your transaction history  
      
    
2.  Select the transaction you would like to review.  
      
    
3.  Select “View on Etherscan” to see the status of the transaction on the blockchain.
