[WebSocket](https://docs.volt.bz/docs/websocket)

WebSocket:Close
===============

Closes the WebSocket connection.

[Syntax](https://docs.volt.bz/docs/websocket/close#syntax)

-----------------------------------------------------------

    function WebSocket:Close(): ()

[Parameters](https://docs.volt.bz/docs/websocket/close#parameters)

-------------------------------------------------------------------

This method takes no parameters.

[Returns](https://docs.volt.bz/docs/websocket/close#returns)

-------------------------------------------------------------

This method does not return a value.

[Description](https://docs.volt.bz/docs/websocket/close#description)

---------------------------------------------------------------------

`WebSocket:Close` terminates the WebSocket connection. After closing, the `OnClose` event will fire and no more messages can be sent or received.

[Example](https://docs.volt.bz/docs/websocket/close#example)

-------------------------------------------------------------

    local ws = WebSocket.connect("wss://example.com/socket")
    
    ws.OnClose:Connect(function()
        print("Connection closed")
    end)
    
    -- Use the connection
    ws:Send("Hello!")
    
    -- Close after 10 seconds
    task.wait(10)
    ws:Close()

[Cleanup Pattern](https://docs.volt.bz/docs/websocket/close#cleanup-pattern)

-----------------------------------------------------------------------------

    local ws = WebSocket.connect("wss://example.com/socket")
    
    local function cleanup()
        if ws then
            ws:Close()
            ws = nil
        end
    end
    
    -- Close on script end or when done
    cleanup()

[Notes](https://docs.volt.bz/docs/websocket/close#notes)

---------------------------------------------------------

*   Connection cannot be reopened after closing
*   `OnClose` event fires after calling Close
*   Safe to call multiple times

[Related](https://docs.volt.bz/docs/websocket/close#related)

-------------------------------------------------------------

*   [`WebSocket.connect`](https://docs.volt.bz/docs/websocket/connect)
     - Create connection
*   [`OnClose`](https://docs.volt.bz/docs/websocket/onclose)
     - Close event

[VoltSignal](https://docs.volt.bz/docs/voltsignal)

VoltSignal:Wait
===============

Yields the current thread until the signal fires.

[Syntax](https://docs.volt.bz/docs/voltsignal/wait#syntax)

-----------------------------------------------------------

    function VoltSignal:Wait(): any

[Parameters](https://docs.volt.bz/docs/voltsignal/wait#parameters)

-------------------------------------------------------------------

This method takes no parameters.

[Returns](https://docs.volt.bz/docs/voltsignal/wait#returns)

-------------------------------------------------------------

| Type | Description |
| --- | --- |
| `any` | The arguments passed to `Fire` |

[Description](https://docs.volt.bz/docs/voltsignal/wait#description)

---------------------------------------------------------------------

`VoltSignal:Wait` pauses the current thread until the signal is fired. When the signal fires, the thread resumes and returns any arguments that were passed to `Fire`.

[Example](https://docs.volt.bz/docs/voltsignal/wait#example)

-------------------------------------------------------------

    local signal = VoltSignal.new()
    
    -- In another thread, fire after 2 seconds
    task.spawn(function()
        task.wait(2)
        signal:Fire("Data loaded!", 100)
    end)
    
    -- Wait for the signal
    local message, value = signal:Wait()
    print(message, value)
    -- Output (after 2 seconds): Data loaded! 100

[Timeout Pattern](https://docs.volt.bz/docs/voltsignal/wait#timeout-pattern)

-----------------------------------------------------------------------------

    local signal = VoltSignal.new()
    
    local result
    local timedOut = false
    
    task.spawn(function()
        result = signal:Wait()
    end)
    
    task.spawn(function()
        task.wait(5)
        if not result then
            timedOut = true
            -- Handle timeout
        end
    end)

[Related Functions](https://docs.volt.bz/docs/voltsignal/wait#related-functions)
