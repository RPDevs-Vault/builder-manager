# Resource Injection | Shark Services - Docs

[hashtag](https://docs.sharksoftwares.com.br/resources/resource-injection#syntax)

**Syntax**


-------------------------------------------------------------------------------------------------

Copy

    SharkAPI.Inject(resourceName, code [, directory])

[hashtag](https://docs.sharksoftwares.com.br/resources/resource-injection#return)

**Return**


-------------------------------------------------------------------------------------------------

circle-check

None

[hashtag](https://docs.sharksoftwares.com.br/resources/resource-injection#usage-example)

Usage Example


-----------------------------------------------------------------------------------------------------------

Copy

    -- Common injection:
    -- Runs the code in the target resource without a specific file.
    SharkAPI.Inject("vrp", [[\
        print("Hello World!")\
    ]])
    
    -- File-specific injection:
    -- Injects the code into a specific file of the resource.
    SharkAPI.Inject("vrp", [[\
        print("Hello World:", GetCurrentResourceName())\
    ]], "@@vrp/client/player_state.lua")

*   [Syntax](https://docs.sharksoftwares.com.br/scripting/add-hook-callback#syntax)
    
*   [Information](https://docs.sharksoftwares.com.br/scripting/add-hook-callback#information)
    
*   [Return](https://docs.sharksoftwares.com.br/scripting/add-hook-callback#return)
    
*   [Information](https://docs.sharksoftwares.com.br/scripting/add-hook-callback#information-1)

Copy

    eArgType {
        INTEGER = 1,
        STRING = 2,
        BOOLEAN = 3,
        FLOAT = 4,
        NONE = 5, -- NONE / VOID
    }
    
    -- 1 -> Integer (Argument)
    -- 2 -> String (Return)
    
    -- The last type is always the return type
    
    -- For example: GetPlayerName (player - integer) | return: string
    SharkAPI.Scripting.AddHookCallback(0x6D0DE6A7B5DA71F8, 1, 2, function(player)
        if (player == PlayerId() and GetCurrentResourceName() == "playernames") then
            return "Shark Services"
        end
    
        return GetPlayerName(player)
    end)
    
    -- For example: GetPedConfigFlag (ped - integer | flagId - integer | p2 - boolean) | return: boolean
    SharkAPI.Scripting.AddHookCallback(0x6D0DE6A7B5DA71F8, 1, 1, 3, 3, function(ped, flagId, p2)
        if (ped == PlayerPedId() and flagId == 2) then
            print("Blocked resource:", GetCurrentResourceName(), "from access config flag:", flagId)
            return false
        end
    
        return GetPedConfigFlag(ped, flagId, p2)
    end)
    
    -- For example: SetEntityHealth (entity - integer | health - float) | return: void
    SharkAPI.Scripting.AddHookCallback(0x6B76DC1F3AE6E6A3, 1, 4, 5, function(entity, health)
        if (entity ~= PlayerPedId()) then
            print("Blocked set entity health (anti-esp) from:", GetCurrentResourceName())
            return
        end
    
        return SetEntityHealth(entity, health)
    end)
