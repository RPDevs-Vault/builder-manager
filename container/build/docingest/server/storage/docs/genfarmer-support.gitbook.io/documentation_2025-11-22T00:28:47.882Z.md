Sau khi bật chế độ nhà phát triển

Để sử dụng chức năng USB Debugging thì cần làm theo các bước như sau:

*   **Vào Cài đặt (Settings):** Tìm biểu tượng Cài đặt (Settings) trên màn hình chính và chạm vào nó.
    
*   **Chọn Giới thiệu về điện thoại (About Phone):** Cuộn xuống và tìm mục Giới thiệu về điện thoại (About Phone) hoặc Giới thiệu về thiết bị (About Device).
    
*   **Chạm vào Số phiên bản (Build Number):** Tìm mục Số phiên bản (Build Number) và chạm vào nó 7 lần liên tiếp. Bạn sẽ thấy một thông báo hiện lên rằng bạn đã trở thành nhà phát triển.
    
*   **Vào lại Cài đặt và kéo xuống dưới cùng**
    
*   **Chọn Tùy chọn nhà phát triển (Developer Options)**
    
*   **Bật chế độ gỡ lỗi USB (USB Debugging):** Tìm mục Gỡ lỗi USB (USB Debugging) và chuyển nút bật tắt sang Bật (On).
    

Tuỳ dòng máy mà có cách bật USB Debugging riêng, thế nên tuỳ vào thiết bị hãy chọn các bật chức năng chuẩn nhất

Trước tiên, để connect thiết bị qua OTG thì cần connect được thiết bị qua cáp USB trước

# API | GenFarmer

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#tai-lieu-api-genfarmer)

Tài liệu API GenFarmer


--------------------------------------------------------------------------------------------------------------------------

Chào mừng bạn đến với tài liệu API của GenFarmer. Tài liệu này giúp bạn hiểu và sử dụng các endpoint RESTful để quản lý hệ thống tự động hóa của GenFarmer.

* * *

### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#gioi-thieu-tong-quan)

🧠 Giới thiệu tổng quan

GenFarmer cung cấp các endpoint để quản lý **Ứng dụng tự động hóa (Automation Apps)**, **Nhiệm vụ (Tasks)** và **Lượt chạy (Runs)** có thể gán cho thiết bị thực hiện tự động.

* * *

### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#url-co-ban)

📦 URL cơ bản

Copy

    http://127.0.0.1:55554/

* * *

### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#nhom-api-ung-dung-tu-dong-hoa)

📁 Nhóm API: Ứng dụng tự động hóa

#### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#danh-sach-ung-dung)

🔍 Danh sách ứng dụng

*   **Phương thức:** `GET`
    
*   **Đường dẫn:** `/automation/apps`
    

**Ví dụ cURL:**

Copy

    curl -X GET 'http://127.0.0.1:55554/automation/apps?userId=3&page=1&limit=25&order=desc&orderBy=updatedAt'

* * *

#### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#chi-tiet-ung-dung)

🔍 Chi tiết ứng dụng

*   **Phương thức:** `GET`
    
*   **Đường dẫn:** `/automation/apps/:id`
    

**Ví dụ cURL:**

Copy

    curl -X GET 'http://127.0.0.1:55554/automation/apps/adoFNVNN6Jwl8FlbZCfni'

* * *

#### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#cap-nhat-ung-dung)

🔧 Cập nhật ứng dụng

*   **Phương thức:** `PUT`
    
*   **Đường dẫn:** `/automation/apps`
    

**Ví dụ cURL:**

Copy

    curl -X PUT 'http://127.0.0.1:55554/automation/apps' \
      -H 'Content-Type: application/json' \
      -d '{"id":"adoFNVNN6Jwl8FlbZCfni","userId":3,"name":"New App","description":"","version":"1.0.0","script":{"flow":{"nodes":[],"edges":[]}}}'

* * *

#### 

[](https://genfarmer-support.gitbook.io/genfarmer/thanh-menu-chinh/api#xoa-ung-dung)

# Guide to connecting OTG in GenFarmer | GenFarmer ENG

**OTG** stands for **On-The-Go**, which is a type of direct connection between the device and your PC. First, to connect the device via OTG, you need to connect it through a USB cable first.

Welcome to the GenFarmer API documentation. This guide helps you understand how to interact with the GenFarmer automation system via RESTful APIs.

[](https://genfarmer-support.gitbook.io/genfarmer-eng/main-menu-bar/api#overview)

🧠 Overview

GenFarmer provides endpoints to manage **Automation Apps**, **Tasks**, and **Runs** that can be assigned to devices for automation scenarios.

* * *

### 

[](https://genfarmer-support.gitbook.io/genfarmer-eng/main-menu-bar/api#base-url)

📦 Base URL

Copy

    http://127.0.0.1:55554/

* * *

### 

[](https://genfarmer-support.gitbook.io/genfarmer-eng/main-menu-bar/api#automation-apis)

📁 Automation APIs

#### 

[](https://genfarmer-support.gitbook.io/genfarmer-eng/main-menu-bar/api#list-apps)

🔍 List Apps

*   **Method:** `GET`
    
*   **Endpoint:** `/automation/apps`
    

**cURL Example:**

Copy

    curl -X GET 'http://127.0.0.1:55554/automation/apps?userId=3&page=1&limit=25&order=desc&orderBy=updatedAt'

🔍 Get App Details

*   **Method:** `GET`
    
*   **Endpoint:** `/automation/apps/:id`
    

**cURL Example:**

Copy

    curl -X GET 'http://127.0.0.1:55554/automation/apps/adoFNVNN6Jwl8FlbZCfni'

#### 

[](https://genfarmer-support.gitbook.io/genfarmer-eng/main-menu-bar/api#update-app)

🔧 Update App

*   **Method:** `PUT`
    
*   **Endpoint:** `/automation/apps`
    

**cURL Example:**

Copy

    curl -X PUT 'http://127.0.0.1:55554/automation/apps' \
      -H 'Content-Type: application/json' \
      -d '{"id":"adoFNVNN6Jwl8FlbZCfni","userId":3,"name":"New App","description":"","version":"1.0.0","script":{"flow":{"nodes":[],"edges":[]}}}'

#### 

[](https://genfarmer-support.gitbook.io/genfarmer-eng/main-menu-bar/api#delete-app)

❌ Delete App

*   **Method:** `DELETE`
    
*   **Endpoint:** `/automation/apps`
    

**cURL Example:**

Copy

    curl -X DELETE 'http://127.0.0.1:55554/automation/apps' \
      -H 'Content-Type: application/json' \
      -d '{"ids": ["2ykTiCbEOAYC_5_tR5Wp-"]}'
