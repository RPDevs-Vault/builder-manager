Delta needs **ROMs** in order to play games. ROMs are special files containing a game's data. There are many ways to get a game's ROM file, but we recommend using a tool like [GB Operator](https://www.epilogue.co/product/gb-operator)
 - though you can find other similar tools online.

Once you have a ROM, you can add it to your library by tapping '+' in the top right corner of Delta's main screen. The ROM must have one of the supported file extensions below.

**Supported File Formats**

System

Accepted Formats

NES

.nes

SNES

.smc, .sfc, .fig

Game Boy

.gb

Game Boy Color

.gbc

Game Boy Advance

.gba

Nintendo DS

.ds, .nds

Nintendo 64

.n64, .z64

Sega Genesis

.gen, .bin, .md, .smd

# Delta Sync | Delta

[PreviousSave States](/using-delta/save-states)

Last updated 1 year ago

Was this helpful?

Delta Sync is a feature that allows you to sync your data (including games, save files, skins, and BIOS files) with either Google Drive or Dropbox.

To enable, go to 'Delta Sync' in your settings and tap on 'Service' to select either Google Drive or Dropbox. Just sign into your account, and Delta will immediately begin syncing your data.

A new 'Sync Status' line will appear in your Delta Sync settings once you have connected a service to help you resolve any issues that may arise.

Delta files are not visible in Google Drive. They are visible in Dropbox, but they are not intended to be edited as this may cause data loss.

**I have AltStore installed, but I still can't download Delta**

If you are having problems downloading Delta, please be sure to check through the AltStore connection .

### 

[](#importing--playing-games)

Importing + Playing Games

**Where do I get games from? I thought I could get them from Delta.**

Unfortunately we are not able to distribute games from Delta directly. We suggest you do your own search to learn about how to get game files. You can see the list of file formats to look out for on our page.

**I put a game in my library, but when I go to play it either doesn't run at all or freezes at certain points. What do I do?**

If everything else is fine, but you are having trouble with running a game, try downloading from another source first to make sure that the file you are trying to run is not altered. Some ROMs may be corrupted and you will run in difficulties while trying to play them with Delta.

If you are attempting to play a "ROM Hack," you may have trouble. While many "ROM Hacks" will run perfectly fine on Delta, some larger hacks may require you to run with additional software.

# Save States | Delta

[PreviousPausing](/using-delta/pausing)
[NextDelta Sync](/using-delta/delta-sync)

Last updated 1 year ago

Was this helpful?

You can access your save states by pausing your game, then pressing either "Save State" (to create a new one) or "Load State" (to load one). You can long-press any save state to view more options.

It's always recommended to use in-game saves as they are more reliable than save states.

### 

[](#making-save-states)

Making Save States

**Select "Save State" from the pause menu,** then tap '+' in the top right to create a new one or tap an existing save state to overwrite it.

### 

[](#loading-save-states)

**Loading Save States**

**Select "Load State" from the pause menu,** then tap the save state you want to load. You can change the ordering of your save states by tapping the arrow in the top right.

You can also access the save states for a game by long-pressing it from the library.

### 

[](#undefined)

### 

[](#renaming-save-states)

Renaming Save States

Long-press the save state you want to rename, then select "Rename".

### 

[](#deleting-save-states)

Deleting Save States

Long-press the save state you want to delete, then select "Delete".

[](#types-of-save-states)

Types of Save States


---------------------------------------------------

### 

[](#auto-save)

Auto Save

These are save states that Delta periodically makes in the background to prevent you from losing your progress.

### 

[](#general)

General

These are save states explicitly made by you and **can** be overwritten.

### 

[](#locked)

**Locked**

These are save states explicitly made by you and **can't** be overwritten. You can lock any save state by long-pressing it and choosing "Lock". To unlock a locked save state, long-press it and choose "Unlock".

[](#preview-save-state)

Preview Save State


-----------------------------------------------

The Preview Save State feature in Delta allows you to choose a save state to load automatically when long-pressing a game. To change the preview state for a game, long-press one of your save states and select "Set as Preview Save State".

Once you set a preview state, you can long-press the game in your library and it will show that save state as a preview. Simply press the game screen to jump right in!

The Hold Buttons feature allows you to keep specific buttons held down so that you can have your fingers free for additional buttons. Some common examples include:

*   Holding down 'A' in Mario Kart to keep on the gas
    
*   Holding down 'B' in Super Mario Bros. to keep Mario running continuously
    
*   Holding down 'A' in Star Fox to keep shooting lasers
    

You can access the Hold Button feature from the Pause Menu to choose buttons to be permanently held down.

Once in the 'Hold Button' screen, tap on the buttons you want held and then tap 'MENU'. When you return, 'Hold Button' will be highlighted, and the buttons you selected will be virtually held down.

You can still press held buttons to perform additional actions (such as launching a fireball as Mario while 'B' is held down to run).

# Customize Mupen64Plus | Delta

[](#overview)

Overview


---------------------------

Delta uses Mupen64Plus for N64 emulation, which means you can customize your N64 experience by following the instructions below.

[](#customizations)

Customizations


---------------------------------------

#### 

[](#resolution)

Resolution

1.  Open the Files app on your iOS device
    
2.  Go to `On My iPhone/Delta/Cores/Mupen64Plus/Config`
    
3.  Open `mupen64plus.cfg` in a text editor
    
4.  Change the values for `ScreenWidth` and `ScreenHeight` under `[Video-General]` to your preferred resolution
    

#### 

[](#texture-packs)

Texture Packs

1.  Open the Files app on your iOS device
    
2.  Place the texture pack in the appropriate directory depending on the type of texture pack
    
    1.  **HTS or HTC:** Add to `On My iPhone/Delta/Cores/Mupen64Plus/cache`
        
    2.  **PNGs:** Add to `On My iPhone/Delta/Cores/Mupen64Plus/hires_texture`
        
    
3.  Open `On My iPhone/Delta/Cores/Mupen64Plus/Config/mupen64plus.cfg` in a text editor
    
4.  Change the value for `txHiresEnable` to 1
    
5.  Open Delta and launch the desired game

If you've ever used an older version of Delta, you may notice that updating to Delta 1.6 hides your old Nintendo DS save states. This is because Delta 1.6 updated the melonDS core to 0.9.5, which unfortunately breaks compatibility with previous save states.

To transfer your save data manually to Delta 1.6, please follow the below instructions.

### 

[](#how-to-migrate-incompatible-save-states)

How to Migrate Incompatible Save States

1.  Launch the game with the save states you wish to transfer
    
2.  Open the Save State screen from the Pause Menu
    
3.  Press "View Incompatible Save States" from the menu button
    
4.  Long-press the save state and choose "Export"
    
5.  Choose where to save the file
    

7.  Import your game into Delta Legacy
    
8.  Launch the game and open the Save State screen from the Pause Menu
    
9.  Tap '+' then choose 'Import Save State'
    
10.  Return to the Pause Menu and choose 'Load State'
    
11.  Choose the imported save state and create a new in-game save
    
12.  Return to the Delta Game Library
    
13.  Long-press the game and choose Manage Save File -> Export Save File
    
14.  Choose where to save the file
    
15.  Launch Delta (1.6)
    
16.  Long-press the same game and choose Manage Save File -> Import Save File
    
17.  Choose the exported save file

# Online Multiplayer | Delta

Delta uses community-hosted Nintendo WFC servers to connect you with other players. Follow the instructions below to start using online features in games like Mario Kart, Pokémon, and more!

[](#how-to-connect)

How to Connect


---------------------------------------

You will only need to perform these steps the first time you connect online.

1.  Attempt to use online features in a Nintendo WFC-compatible DS game.
    
2.  Tap "Choose Server" on the alert that appears.
    
3.  Choose a server from the list or provide a custom DNS.
    

You can change your server later by going to Delta Settings > Core Settings (Nintendo DS) > WFC Server, but you may need to re-register your friend codes.

1.  Tap "Done" in the top right.
    
2.  Restart your game by pressing "Restart" on the alert that appears.
    
3.  If you are using BIOS Files, you'll need to erase your Nintendo WFC Configuration in-game
    
    1.  Access the Nintendo WFC settings of the game. This is dependent on the game, but can usually be found from the game's main menu.
        
    2.  Go to **Options,** then tap "Erase Nintendo WFC Configuration"
        
    

[](#troubleshooting)

Troubleshooting


-----------------------------------------

1.  Devices on the same Wi-Fi network may not be able to connect to each other. We recommend switching to cellular data (if available) as a workaround.
    
2.  Make sure you have chosen a server in Delta Settings > Core Settings (Nintendo DS) > WFC Server.
    
3.  If you're using a VPN or proxy, make sure to disable them. Otherwise you may be temporarily banned from using servers like Wiimmfi.
    
4.  Try erasing your Nintendo WFC Configuration in-game
    
    1.  Access the Nintendo WFC settings of the game. This is dependent on the game, but can usually be found from the game's main menu.
        
    2.  Go to **Options,** then tap "Erase Nintendo WFC Configuration"
        
    
5.  Consider deleting your BIOS files by going to iOS Settings → Storage → Delta, then deleting `bios7.bin`, `bios9.bin`, and `firmware.bin`.
    

Removing BIOS files will prevent you from using the DS Homescreen and dual-slot functionality for GBA games.

[](#wfc-error-codes)

WFC Error Codes


-----------------------------------------

**20105 (Login Error)**

If you encounter this error, first try choosing a different WFC server in Delta Settings > Core Settings (Nintendo DS) > WFC Server. If you still have trouble connecting, try using an alternative WiFi network/cellular.

**20110 (ISP Issue)**

If you encounter this error, first try choosing a different WFC server in Delta Settings > Core Settings (Nintendo DS) > WFC Server. If you still have trouble connecting, try using an alternative WiFi network/cellular.

**23917 (Banned)**

If you see this error, you may need to erase your Nintendo WFC Configuration in-game. You will need to find the Nintendo Wi-Fi Connection Setup page for your game and then go to Options > Erase Nintendo WFC Configuration.

**50199 (Connection Interrupted)**

Try checking your connection stability, using another WiFi Network, or switching to cellular.

**51099 (Connection Error)**

If you see this error, you may need to go to your Core Settings > Nintendo DS and reset your Online Settings before connecting again.

**52000 (No Connection Found)**

You may need to try another WiFi/Cellular network or reset your router.

**52100 (DNS Error)**

You will see this error if your DNS settings are incorrect. If you are using a custom server, check that you have entered the correct DNS settings.

**52200 (Connection Error)**

If you encounter this error, first try choosing a different WFC server in Delta Settings > Core Settings (Nintendo DS) > WFC Server. If you still have trouble connecting, try using an alternative WiFi network/cellular.

**60000 (Switched Servers)**

You will see this error if you switch servers while connecting to Nintendo WFC. You should restart your game and try again.

**61020 (Maintenance Error)**

You will see this error if the server you are connecting to is under maintenance. You can try connecting to another server or trying again later.

**80430 (Same WiFi)**

You may encounter this error if using the same WiFi with multiple devices. Try connecting each device to a different WiFi network/cellular.

**81010 (Disconnected)**

Due to an unknown error, you were disconnected from Nintendo WFC and should try again.

**86420 (DNS Error)**

This may be due to an unexpected error with the WiFi you are attempting to connect to. Try using an alternate WiFi network/cellular or resetting your router.

**91010 (Banned Firmware/Kicked from Server)**

You were kicked from the server or your DS Firmware is banned on the server you are connected to. You can try again, use alternate BIOS files, or connect to a new server.
