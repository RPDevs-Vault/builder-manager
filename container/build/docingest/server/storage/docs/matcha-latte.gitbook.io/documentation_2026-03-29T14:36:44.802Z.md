# Getting Started | Matcha

This is the documentation for the Matcha LuaVM. Please note that the **LuaVM is not an executor, it does not internally hook any functions in the Roblox engine**. Instead it tries to emulate those functions externally. This makes it completely undetected, but also means it lacks direct access to internal Roblox API functions.

### 

[hashtag](https://matcha-latte.gitbook.io/matcha#resources)

Resources

To understand how the LuaVM works, you first need to understand the basics of Lua(U). For that, visit lua.org

    local square = Drawing.new("Square")

# Getting Started | Matcha

This is the documentation for the Matcha LuaVM. Please note that the **LuaVM is not an executor, it does not internally hook any functions in the Roblox engine**. Instead it tries to emulate those functions externally. This makes it completely undetected, but also means it lacks direct access to internal Roblox API functions.
