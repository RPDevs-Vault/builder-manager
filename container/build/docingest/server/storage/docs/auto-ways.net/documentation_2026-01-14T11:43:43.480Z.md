[Qu’est-ce que le paramètre « demarche » ?](https://auto-ways.net/ufaq/quest-ce-que-le-parametre-demarche/)

------------------------------------------------------------------------------------------------------------

Le paramètre « demarche » correspond au type d’opération administrative que vous souhaitez effectuer. Il peut s’agir d’un changement de titulaire, d’une première immatriculation, d’un duplicata ou d’un changement d’adresse. Chaque démarche implique des frais différents.

*   [auto-ways-network](https://auto-ways.net/author/auto-ways/ "Articles par auto-ways-network")
    
*   14/07/2025

*   [Carte grise](https://auto-ways.net/ufaq-category/carte-grise/)
    

[L’API couvre-t-elle les DOM-TOM ?](https://auto-ways.net/ufaq/lapi-couvre-t-elle-les-dom-tom/)

------------------------------------------------------------------------------------------------

Oui, notre API couvre l’ensemble du territoire français, y compris les départements et régions d’outre-mer (DROM) et les collectivités d’outre-mer (COM). Les barèmes spécifiques à ces territoires sont intégrés et mis à jour régulièrement.
