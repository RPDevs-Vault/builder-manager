# Virtualization. | Busan.SU Docs

[](https://busan-su.gitbook.io/busan.su-docs/windows-steps/program/bios/virtualization.#virtualization)

Virtualization


---------------------------------------------------------------------------------------------------------------------------

VIRTUALIZATION MUST BE ENABLED

How to enable virtualization

*   Restart your computer.
    
*   Log in to BIOS/UEFI.Press the key (usually F2, Delete, Esc or F10) immediately after turning on the device.
    
*   Go to the processor settings.Find a section that can be called "Advanced", "CPU Configuration", "Processor" or similar.
    
*   Enable virtualization. Find an option such as "Intel VT", "Intel Virtualization Technology", "AMD-V" or "SVM Mode" and change the value to "Enabled".
    
*   Save the changes. This can usually be done by pressing F10 and then selecting "Yes" to confirm.
    
*   Restart your computer.

# UEFI | Busan.SU Docs

[](https://busan-su.gitbook.io/busan.su-docs/windows-steps/program/bios/uefi#uefi)

UEFI


--------------------------------------------------------------------------------------------

Be sure to install UEFI !

A short guide for installing the uefi bios.

**Step 1: Prepare**

*   Save your work.
    
*   Restart the PC.
    

**Step 2: Enter BIOS/UEFI**

*   Press the key to enter BIOS. Upon restarting the system, you often need to press one of the following keys (usually F2, Delete, Esc, or F10). A prompt may appear on the screen indicating which key to press.
    
*   Open the BIOS/UEFI settings. If you successfully pressed the correct key, you will enter the BIOS menu.
    

**Step 3: Find UEFI Settings**

*   Look for the "Boot" or "Boot Options" tab. In some systems, UEFI settings may be located in different sections, such as "Advanced" or "System Configuration."
    
*   Select the boot mode. Find the option related to the boot mode (usually called "Boot Mode," "UEFI/Legacy Boot," or something similar).
    

**Step 4: Change Mode to UEFI**

*   Change the mode to UEFI. If the "Legacy" or "CSM" option is selected, change it to "UEFI."
    
*   Save the changes. Typically, this requires pressing the F10 key and selecting "Yes" to confirm.
    

**Step 5: Reboot and Verify**

*   Restart the computer. After saving the settings, the system will reboot.
    
*   Check the boot settings. Ensure that the system is booting in UEFI mode. You can do this by entering “msinfo” in the start menu and checking the "BIOS mode" tab.
    

**Tips**

*   If BIOS/UEFI does not open, try using different keys during boot or check the documentation on your device manufacturer's website.

# TPM | Busan.SU Docs

[](https://busan-su.gitbook.io/busan.su-docs/windows-steps/program/bios/tpm#ftpm-tpm)

Ftpm / Tpm


-----------------------------------------------------------------------------------------------------

Ftpm / Tpm is usually disabled (not for valorant)!

How to disable TPM

*   Restart your computer.
    
*   Enter BIOS/UEFI and press the key (usually F2, Delete, Esc or F10) immediately after turning on the computer.
    
*   Find the TPM section. Look for sections named "Security", "Trusted Computing" or "TPM".
    
*   Turn off the TPM. Find the option associated with the TPM and set the value to "Disabled".
    
*   Save the changes. This is usually done using the F10 key to save the settings and restart.
    
*   Restart your computer. After exiting BIOS/UEFI, the system will restart.

Secure Boot


---------------------------------------------------------------------------------------------------------------

Secure boot is usually disabled (not for valorant)!

How to disable Secure Boot

*   Restart your PC.
    
*   Enter BIOS/UEFI: Press F2, Delete, Esc or F10 (depending on your device).
    
*   Find the section with Secure Boot: Usually in "Security" or "Boot".
    
*   Disable Secure Boot: Set the value to "Disabled".
    
*   Save the changes: Press F10 and confirm.
