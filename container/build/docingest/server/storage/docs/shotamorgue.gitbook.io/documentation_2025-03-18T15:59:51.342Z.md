[](#function-chartno)

Function Chartの見方


--------------------------------------------

**Button**の行は対象のボタンを表します。各ボタンで実際に操作するパラメータはモード、コマンドごとに異なるため、この表ではその対応を表しています。

### 

[](#mode)

Mode

**Mode**の行は各モード中の操作について表します。**\[PLAY\]/\[REC\]**を押すたびに切り替わる**STOP/PLAY/REC**の3つの**モード**があり、操作中は常にいずれかの状態です。

### 

[](#command)

Command

**Command**の行は各コマンドボタンを押している間の操作について表します。**STOPモード**もしくは**PLAYモード**の状態で**\[SOUND\]/\[FUNC\]**を押すと、押している間はそれぞれコマンド入力が可能になります。

### 

[](#li)

例

**STOPモード**時の操作方法を見ていきます。

*   **\[PLAY\]**を押すと、**STOPモード**から**PLAYモード**に切り替わります。
    
*   **\[REC\]**を押すと、**STOPモード**から**RECモード**に切り替わります。
    
*   **\[SOUND\]**を押している間、**SOUNDコマンド**の入力が可能です。
    
*   **\[FUNC\]**を押している間、**FUNCコマンド**の入力が可能です。
    
*   **\[SOUND\]+\[FUNC\]**を押している間、**FXコマンド**の入力が可能です。
    
*   **\[1\]/\[2\]/\[3\]/\[4\]**は鍵盤として機能します。
    
*   **\[A\]/\[B\]**を押すと**Octave/Tune**パラメータが増減します。
    

![](https://shotamorgue.gitbook.io/~gitbook/image?url=https%3A%2F%2F3944720961-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LYie2vvd17-4faFVDfq%252F-MFzmEzAjFTUfltGgJcI%252F-MG-tMjXR3JbkQu8OnLO%252Fn0x_pd_fn_chart.png%3Falt%3Dmedia%26token%3De1061cf3-b7df-45ba-bf66-004774c06803&width=768&dpr=4&quality=100&sign=32a60c05&sv=2)

# 3. パラメータを調節する | ParipiDestroyer Manual

ここでは**Volume**を例に各種パラメータを調節する**コマンド入力**の方法を解説します。

**STOPモード**もしくは**PLAYモード**の状態で**\[FUNC\]**を押しながら**\[A\]/\[B\]**を押すと**Volume**を調節することができます。

[](#id-1-nowosuru)

1\. 現在の音量を確認する


--------------------------------------

**\[FUNC\]**を押している間、**\[A\]/\[B\]**の上部にあるランプ**<A>/<B>**は現在の**Volume**の値を示します。

[](#id-2-wosakusuru)

2\. 音量を小さくする


--------------------------------------

**\[FUNC\]**を押しながら**\[A\]**を押すと**Volume**が一段階小さくなります。

**\[FUNC\]**を押し続けながら**\[A\]**を押す操作を繰り返すと**Volume**を連続して小さくすることができます。

[](#id-3-wokikusuru)

3\. 音量を大きくする


--------------------------------------

**\[FUNC\]**を押しながら**\[B\]**を押すと**Volume**が一段階大きくなります。

**\[FUNC\]**を押し続けながら**\[B\]**を押す操作を繰り返すと**Volume**を連続して大きくすることができます。

他のパラメータを調節したい場合は、[**機能表**](https://shotamorgue.gitbook.io/paripidestroyer/intro/quick-chart/function-chart)
を参考に対象のモード/ボタンで同様の操作を行なってください。

# 表記規則 | ParipiDestroyer Manual

[](#botannobu)

ボタン・ノブ


--------------------------

*   ボタン・ノブは太字で表記し、括弧で囲みます。
    
    *   例: PLAYボタン、押すたびに機能する (ラッチ動作) -> **\[PLAY\]**
        
    *   例: Cutoffノブ -> **\[Cutoff\]**
        
    
*   押し続ける必要があるボタンはさらに括弧で囲みます。
    
    *   例: SOUNDボタンを長押し、押している間だけ機能する (モーメンタリ動作) -> **\[\[SOUND\]\]**
        
    
*   指定時間や回数指定がある場合は後に時間を括弧で表記します。
    
    *   例: PLAYボタンを1秒間押し続ける -> **\[\[PLAY\]\](1sec.)**
        
    *   例: RECボタンを押して離す動作を2回繰り返す -> **\[REC\](x2)**
        
    

[](#ranpu)

**ランプ**


-----------------------

太字で表記し、角括弧で囲みます。

例: PLAYランプ -> **<PLAY>**

[](#mdokomando)

**モード・コマンド**


---------------------------------

太字かつ大文字で表記します。

例: STOPモード -> **STOPモード**

[](#paramta)

**パラメータ**


---------------------------

太字で表記します。

例: Resonance -> **Resonance**

[](#she-ding-zhi)

**設定値**


------------------------------

太字かつ斜体で表記します。

例: Bypass -> _**Bypass**_

[](#and)

**AND**


---------------------

複数を同時に扱う場合はプラスで区切り併記します。 また順序を区別し、左から右の順で扱います。

例: SOUNDボタン押しながら1ボタンを押す -> **\[SOUND\]+\[1\]**

[](#or)

OR


---------------

複数のうちいずれかを扱う際はスラッシュ記号で区切り併記します。

例: SOUNDボタンもしくはFUNCボタンを押す -> **\[SOUND\]/\[FUNC\]**

[](#lie-ju)

列挙


-------------------

3つ以上の項目を列挙する際は**PLAY/REC/SOUND/FUNC/1/2/3/4/A/B/Cutoff/Decay**の順で中略します。

**例:** **<PLAY>**から**<B>**までのランプ → **<PLAY>~<B>**

# 音声出力 | ParipiDestroyer Manual

[](#supk)

内蔵スピーカー


----------------------

**内蔵スピーカー**は**ヘッドフォン端子**に何も繋いでいない状態で有効になります。

電池駆動時はスピーカーの音量が電池の電圧によって段階的に制限されます。USB電源接続時は制限されません。

鳴らす音色によっては音割れが起こる場合があります。その際は、音割れしない音量まで下げてお使いください。

[](#heddofonrain)

ヘッドフォン/ライン出力


-----------------------------------

*   **コネクタ形状:** 3.5mm TRS ステレオミニジャック
    
*   **用途:** ヘッドフォンもしくはLINE入力を持つ外部機器に接続
    
*   **機能:**
    
    *   ステレオ音声出力
        
    *   スピーカー自動OFF
        
        *   ステレオミニプラグを差し込むと内蔵スピーカーOFF
            
        
    *   Sync信号出力
        
        *   スプリットケーブルを用いて片方からSync信号、もう片方からモノラル音声を出力
            
        
    

**ヘッドフォン端子**にステレオミニプラグを接続すると、音声の出力が内蔵スピーカーからヘッドフォン/ライン出力に切り替わります。

内蔵スピーカーとヘッドフォン/ライン出力を同時に鳴らすことはできません。

ヘッドフォンを接続する場合は音量にご注意ください。

[](#yin-liang-diao-jie)

音量調節


---------------------------------

音量は**STOP/PLAYモード**時に**\[FUNC\]**を押しながら**\[1\]/\[2\]**で調整します。

*   **\[FUNC\]**を押しながら**\[1\]**を押すと音量が一段階小さくなります
    
*   **\[FUNC\]**を押しながら**\[2\]**を押すと音量が一段階大きくなります
    

内蔵スピーカー有効時はスピーカーの音量、ヘッドフォン/ライン出力時はヘッドフォン/ライン出力の音量とそれぞれの状態で音量を調節します。

スピーカーの音量設定値とヘッドフォン/ライン出力の音量設定値はそれぞれ個別に扱われます。

設定した音量は**Save**操作もしくは**Auto Save**によって保存されます。

ParipiDestroyer | Crazy Synth Kids


--------------------------------------------------------------------------------

パリピデストロイヤーは16ステップのシーケンサー、モノフォニックのベースシンセ音源、ディレイ&ディストーションエフェクトが一体になった手のひらサイズのコンパクトなマシンです。乾電池一本で動作し、スピーカーを内蔵しているためいつでもどこでもプレイすることができます。

[](#ji-ben-shi-yang)

基本仕様


------------------------------

*   **音源構成:** 1VCO/1VCF/1VCA/2EG (バーチャルアナログ/モノフォニック)
    
*   **波形:** Triangle/Saw/Octave Saw/Square Saw/Square1/Square2/Square3/PWM
    
*   **エフェクト:** Distortion/Delay
    
*   **鍵盤:** 4鍵 (ピッチ編集可能)
    
*   **シーケンサー:** 16ステップ/4パターン
    
*   **出力端子:** ヘッドホン/LINE出力端子 (3.5mm ミニジャック)
    
*   **入力端子:** SYNC端子 (3.5mm ミニジャック)
    
*   **電源:** 単4アルカリ乾電池 (1.5V) x1もしくはUSB-C端子 (5V/500mA)
    
*   **消費電力:** 最大1.25W
    
*   **サイズ:** W64mm x L54mm x H38mm
    
*   **重量:** 48g(本体のみ)/59g(本体+電池)

# メモリ | ParipiDestroyer Manual

[](#memori)

メモリ構成


----------------------

本体内のメモリは現在の状態を保持する**SRAM**とデータを保持し続ける**FLASH**の二種類があります。**SRAM**のデータは電源の供給を断つと失われますが、**FLASH**に保存したデータは保持されます。

設定したパラメータや記録したシーケンスは**SRAM**に保持されています。データを**FLASH**に保存せずに電池とUSBケーブル両方を外すと、**SRAM**上のデータは失われます。

[](#shkensuparamtano)

シーケンス/パラメータの保存


-----------------------------------------

# トップパネルの交換 | ParipiDestroyer Manual

パリピデストロイヤーのトップパネルは、ネジを二箇所外すことで取り外しができます。標準で装着されているパネルはリバーシブルになっており、お好きなデザインを選ぶことができます。

### 

[](#namono)

必要なもの

プラスドライバー (PH1/NO.1)

一般的なプラスドライバーを使用できますが、質の悪いものや規格が合わないものを使用するとネジ穴を痛める可能性があります。

必要以上にきつく閉めると基板が歪んだり、割れたりする可能性があります。また、緩すぎると使用中に分解してしまう可能性があります。

### 

[](#jiao-huan-shou-shun)

交換手順

1.  本体を抑えた状態で、ねじを緩めて外します。
    
2.  本体横を掴みつまみを下に向かって押さえつけながらトップパネルをゆっくりと外します。
    
3.  交換するトップパネル/裏返したトップパネルを装着し、2箇所のねじを締めます。

# RECモード | ParipiDestroyer Manual

RECモード概要


-------------------------

**RECモード**ではステップ入力によってシーケンスパターンの作成・編集ができます。

[](#play-shkensuno-play)

PLAY (シーケンスの再生) | \[PLAY\]


--------------------------------------------------------

**RECモード**中に**\[PLAY\]**を押すと**RECモード**は終了して**PLAYモード**になり、シーケンスの再生が開始します。シーケンスの途中で**\[PLAY\]**を押すと、押した時点でのステップが再生されるステップ数になります。

[](#prev-stepnext-step-ksoru-recsound)

Prev. step/Next Step (カーソル移動) | \[REC\]/\[SOUND\]


---------------------------------------------------------------------------------------------

**\[REC\]**を押すと現在のステップ位置が減少方向に、**\[SOUND\]**を押すと増大方向に移動します。また、0ステップを下回ると16ステップに、16ステップを上回ると0ステップに移動します。

[](#slide-accent-or-func)

Slide/Accent | \[FUNC\]


------------------------------------------------------

1.  **\[FUNC\]**を押すと**<Decay>**が点灯し、現在のステップの**Slide**を有効化します。_**Slide**_は、現在のステップから次のステップの音程までをなめらかにつなぎます。
    
2.  **\[FUNC\]**をもう一度押すと**<Decay>**が消灯して**<Cutoff>**が点灯し、現在のステップの**Slide**を無効化、**Accent**を有効化します。**Accent**が入ったステップは、音量が少し大きくなります。
    
3.  **\[FUNC\]**をさらにもう一度押すと、**<Cutoff>**インジケーターが消灯し、現在のステップ**Accent**入力を無効化します。

Octave | \[A\]/\[B\]

[](#q4-usbkburuwoideirunonigawarezugachiteshimaimasunazedeshouka)

Q4. USBケーブルを繋いでいるのに電源供給が行われず、電源が落ちてしまいます。なぜでしょうか？


------------------------------------------------------------------------------------------------------------------------

本製品は最低20mA-最大500mA程度の電流を消費します。モバイルバッテリーなど一部の製品では、一定以下の電流時に給電を遮断する機能 (オートパワーオフ) が付いています。これらは、スマートフォンなどの充電のために設計されており、充電完了後に無駄に電力を消費しないようにするためのものです。 本製品の消費電流は通常この機能の閾値を下回るため、供給が遮断されてしまいます。

**解決策:** モバイルバッテリーにUSBポートが複数ある場合、別の消費電流が大きな製品を接続することでオートパワーオフを回避することができます。 また、本製品の内蔵スピーカーである程度の音量を出すことで消費電流が大きくなり、オートパワーオフを回避することができます。

### 

[](#usbgananitsuite)

USB給電が可能な製品について

USB電源アダプタを利用する際は、以下の点を守って使用してください。

*   必ずメーカーが指定する電圧のコンセントに差し込んで使用してください。
    
*   プラグに埃や汚れ、液体が付着している場合は拭き取ってから使用してください。
    
*   他の電気機器の電源コードと一緒にタコ足配線をしないでください。また、 本製品の定格消費電力に合ったコンセントに接続してください。
    
*   濡れる可能性のある場所（洗面台、浴室、シャワー室付近など）では使用しないでください。また、濡れた手で電源アダプタを抜き差ししないでください。
    

また、以下のいずれかの条件に該当する場合は、電源アダプタおよびケーブルの使用を中止してください。

*   電源アダプタのプラグまたはピンが損傷している。
    
*   充電ケーブルがすり切れている、またはその他の損傷がある。
    
*   電源アダプタが湿気の多い場所にあった、または電源アダプタ内部に液体が入った。
    
*   電源アダプタを落として筐体が破損した。
    
*   製品に異常が発生したり、製品内部に異物が入ったりした場合は直ちに使用をやめてUSB電源アダプタをコンセントから抜いてください。
    

### 

[](#richiumuionbatterwoshitanitsuite)

リチウムイオンバッテリーを内蔵した製品について

バッテリーを自分で交換しようとしないでください。不適切に交換または修理すると、バッテリーが損傷して過熱や負傷を引き起こすおそれがあります。不要になったバッテリーは、リサイクルするか、家庭のごみとは分けて廃棄してください。バッテリーを焼却処分しないでください。

### 

[](#no)

聴覚の損傷

大音量で長時間使用すると、聴覚を損なうおそれがあります。耳鳴りや聴力の低下を感じた場合は直ちに使用をやめ、医師に相談してください。音を鳴らす際は、次の手順に従ってください。

*   **ヘッドフォン接続時**
    
    *   ヘッドフォンに接続してお使いの際は機器本体の音量を十分に下げてから接続し、音を鳴らして音量が適切であることを確認してからゆっくりと耳に装着してください。
        
    
*   **その他外部機器接続時**
    
    *   スピーカーやレコーダー、オーディオインターフェイスなどの外部機器に接続してお使いの際は、機器本体と外部機器の音量を十分に下げてから接続し、音を鳴らしながらゆっくりと適切な音量までボリュームをあげてください。

交換方法

1.  電源をOFFにします。
    
2.  本体を裏返し、本体を押さえながら右側のプラス極側に爪を引っ掛けて電池を外します。
    
3.  極性を確認しながら新しい乾電池を装着します。
    

本体に充電機能は有りません。

電池装着部に記載された正しい極性で電池を装着してください。極性を間違えると機器は動作せず、電池の寿命が短くなったり液漏れする可能性があります。

2本以上の電池を用いた製品の場合、同種の新しい電池のみを使用し、別種の電池や新しいものと古いものを混ぜて使用しないでください。

電池を取り外すと保存されていないシーケンスやパラメータは失われます。手順1で電源をOFFにする前にSave動作を行なってください。

液漏れを防ぐため、長期間使用しない場合は電池を取り外してください。

使用済み電池の廃棄については、お住いの地域で指定された方法に従ってください。

モバイルバッテリーは多くの場合、消費する電流が少ないと電源供給を遮断するオートパワーオフ機能がついています。このため、本製品をモバイルバッテリーで連続使用する際は内蔵スピーカーを駆動し、なるべく電流を消費する必要があります。

もしくはオートパワーオフ機能を持たないモバイルバッテリーを使用してください。

1\. RECモードに切り替える

**STOPモード**の状態で**\[REC\]**を押すと**RECモード**に切り替わります。

シーケンスが再生されている場合は**PLAYモード**です。**\[PLAY\]**を押して**STOPモード**に切り替えてから**\[REC\]**を押してください。

2\. シーケンスを記録する


-------------------------------------------

**RECモード**中**\[1\]~\[4\]**はシーケンス記録用の**Keyboard**として機能し、1つ押す度に1ステップずつシーケンスが入力されていきます。**\[PLAY\]**を押すか、**入力が16ステップに達する⁽\*¹⁾**と**RECモード**は終了しシーケンスの再生が開始します。

**\*1 Global Options**の**AutoPlay**が_**ON**_の場合のみ。

[](#id-3-octavewosuru)

3\. Octaveを入力する


-------------------------------------------

**RECモード**でのシーケンス入力時に、**\[A\]**を押すと1オクターブ下、**\[B\]**を押すと1オクターブ上が選択されます。続いて**Keyboard**押すと、選択したオクターブでシーケンスが入力されます。

**Octave**を入力している間ステップは進まず、**Keyboard**を押した時にステップが進みます。
