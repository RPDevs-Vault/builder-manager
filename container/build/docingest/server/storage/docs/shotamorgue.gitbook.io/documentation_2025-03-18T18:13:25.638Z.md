ParipiDestroyer | Crazy Synth Kids


--------------------------------------------------------------------------------

パリピデストロイヤーは16ステップのシーケンサー、モノフォニックのベースシンセ音源、ディレイ&ディストーションエフェクトが一体になった手のひらサイズのコンパクトなマシンです。乾電池一本で動作し、スピーカーを内蔵しているためいつでもどこでもプレイすることができます。

[](#ji-ben-shi-yang)

基本仕様


------------------------------

*   **音源構成:** 1VCO/1VCF/1VCA/2EG (バーチャルアナログ/モノフォニック)
    
*   **波形:** Triangle/Saw/Octave Saw/Square Saw/Square1/Square2/Square3/PWM
    
*   **エフェクト:** Distortion/Delay
    
*   **鍵盤:** 4鍵 (ピッチ編集可能)
    
*   **シーケンサー:** 16ステップ/4パターン
    
*   **出力端子:** ヘッドホン/LINE出力端子 (3.5mm ミニジャック)
    
*   **入力端子:** SYNC端子 (3.5mm ミニジャック)
    
*   **電源:** 単4アルカリ乾電池 (1.5V) x1もしくはUSB-C端子 (5V/500mA)
    
*   **消費電力:** 最大1.25W
    
*   **サイズ:** W64mm x L54mm x H38mm
    
*   **重量:** 48g(本体のみ)/59g(本体+電池)
