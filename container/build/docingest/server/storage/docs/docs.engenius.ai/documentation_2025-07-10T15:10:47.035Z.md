# Mobile App | Cloud User Manual

The EnGenius Mobile App is a mobile user interface (UI) for EnGenius Cloud. You can keep an eye on your network when you are on the go. This is a great solution for around the clock network support. Versions are available for Android on Google Play and iOS via the App Store.

[](https://docs.engenius.ai/engenius-cloud/mobile-app#adding-a-device)

Adding a Device


-------------------------------------------------------------------------------------------

This section explains how to add a networking device to your network using the EnGenius Mobile App.

1\. Navigate to the **Inventory** tab and tap the **\+** symbol on the bottom-right of the screen.

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKc69ElkYWpAi0IbKqX%252F-MKc8MGRdtP63gEohmy2%252F1603769979459.jpg%3Falt%3Dmedia%26token%3Dbc8f0c96-e6f1-4e45-8323-083ae9465024&width=768&dpr=4&quality=100&sign=df302ef0&sv=2)

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKc69ElkYWpAi0IbKqX%252F-MKc8SrR51fs73haFjlr%252F1603770004831.jpg%3Falt%3Dmedia%26token%3Dcf3e9038-6dad-493b-aed0-df5eabe59f95&width=768&dpr=4&quality=100&sign=a28d56f6&sv=2)

2\. Find the QR code at the bottom of the device and scan it.

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-LnurgsWeBcaTtH8jr_r%252F-Lnurs4x6QZlmomt-amn%252F1567576903246.jpg%3Falt%3Dmedia%26token%3D16411116-ed48-4b9f-ba7d-114476c5851c&width=768&dpr=4&quality=100&sign=9912bbae&sv=2)

If the camera successfully scanned a QR code, the app will display the Device Information. You could tap

**Register** to complete the Registration.

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKc9GyhC7jopKglZbE_%252F-MKcmRynNXQqOGCxqwgU%252FS__3538954.jpg%3Falt%3Dmedia%26token%3Dffa4b396-c7f5-4733-b1bb-7b138a904aea&width=768&dpr=4&quality=100&sign=b9a21042&sv=2)

If you failed to scan the QR code successfully, you could tap **Okay, try another**.

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKcnPqxCfW-aOmpA375%252F-MKcokwWqpyFbnSGo3bY%252F6B72461E-DB94-43AD-9291-57D45D735CAA.png%3Falt%3Dmedia%26token%3Df38ddf57-071d-4257-b0e0-d1fd5f254aa2&width=768&dpr=4&quality=100&sign=b44c7c54&sv=2)

3\. Once a device has been successfully registered, you can tap **Assign to Network site now.**

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKc9GyhC7jopKglZbE_%252F-MKcmbwHtpm_HULdUxXB%252FS__3538953.jpg%3Falt%3Dmedia%26token%3Da41427da-16ca-432f-9150-78344b9066a2&width=768&dpr=4&quality=100&sign=56898c23&sv=2)

4\. Tap the Network and tap **Apply**.

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKc9GyhC7jopKglZbE_%252F-MKcm7paVm8ysosZ-fx-%252FS__3538951.jpg%3Falt%3Dmedia%26token%3D1dc647a9-d3c5-450a-b4e8-57ea2f8c1c4c&width=768&dpr=4&quality=100&sign=333959e1&sv=2)

5\. Once you select the wrong network, you could tap **Change** to select the correct network**.** If the network is correct, tap **Next .**

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKcqg2f4G--Z0ZvwHJH%252F-MKcr_kCbgHEMSJy8jsh%252FS__3538950_0.jpg%3Falt%3Dmedia%26token%3Dccbe6195-c61d-457e-8d16-0b2f13f43545&width=768&dpr=4&quality=100&sign=9d7d7c2&sv=2)

**6\.** You could tap **Finish** to complete the whole process or tap **Register more** to register other devices .

![](https://docs.engenius.ai/engenius-cloud/~gitbook/image?url=https%3A%2F%2F3893603398-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-LgVsoqihVR3I0VRYmyO%252F-MKcqg2f4G--Z0ZvwHJH%252F-MKcr5pwTDjQjUsKGwU9%252FS__3538948.jpg%3Falt%3Dmedia%26token%3D4f059652-129c-45d3-9dac-999c51cb8084&width=768&dpr=4&quality=100&sign=c78ec54e&sv=2)
