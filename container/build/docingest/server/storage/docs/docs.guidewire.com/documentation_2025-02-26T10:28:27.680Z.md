# Introduction | IS Deployments API

[Skip to main content](#__docusaurus_skipToContent_fallback)

Use the IS Deployments API to deploy the InsuranceSuite apps. You can perform the following operations:

*   Deploy all or selected InsuranceSuite apps.
*   Check a planet status.
*   Get a list of branches for a specified star system, and planet.
*   Get a list of builds for a specified star system, planet, and branch.
*   Check if rollback is available.
*   Perform a rollback.
*   Restore a database backup on a planet.
