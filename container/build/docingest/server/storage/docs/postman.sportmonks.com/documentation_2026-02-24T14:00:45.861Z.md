API3
====

Our Football API 3.0 is a REST API that can be used to request football data from our wide-ranging database and has the following characteristics:

*   Highly customizable requests
    
*   Fast livescores
    
*   Reliable and accurate data
    

In addition to the [Football API](https://docs.sportmonks.com/football/tutorials-and-guides/tutorials/www.sportmonks.com/football-api)
, Sportmonks offers a [Prediction API,](https://www.sportmonks.com/football-api/football-predictions/)
 [Expected Goals](https://www.sportmonks.com/football-api/xg-data/)
 data, a [Premium Odds Feed](https://www.sportmonks.com/football-api/premium-odds-feed/)
, a [News API](https://www.sportmonks.com/football-api/football-news-api/)
, and multiple [Football Widgets](https://www.sportmonks.com/football-api/livescore-widget/)
.

The Football API covers over 2,300 leagues, and over 1,350 leagues are covered in the Prediction API. Every continent, country, league, season, stage, round, venue, TV Station, player, coach, referee, bookmaker, market, news article, event, and fixture has its own ID.

Head over to our [Getting Started section](https://docs.sportmonks.com/football/welcome/making-your-first-request)
, where you'll create your own API token and create your first request
