Comment créer des clés API sur Binance ?
========================================

Publié le 2018-04-09 03:17

_Dernière mise à jour : le 20 mars 2025_

La création d’une API vous permet de vous connecter aux serveurs de Binance via plusieurs langages de programmation. Les données peuvent être extraites de Binance et interagir avec des applications externes. Vous pouvez consulter les données de votre portefeuille et de vos transactions, effectuer des trades, et même déposer ou retirer vos fonds en utilisant des programmes conçus par des tiers.

**Veuillez remarquer qu’avant de créer une clé API, vous devez effectuer un dépôt de n’importe quel montant sur votre portefeuille Spot pour activer votre compte et** [**effectuer la vérification d’identité**](https://www.binance.com/fr/support/faq/360027287111)
**.** 

Pour en savoir plus sur l’API de Binance, veuillez vous référer à cette [documentation](https://developers.binance.com/fr/)
.

### Guide détaillé pour créer sa propre clé API Binance

**(Site Web) Comment créer sa propre clé API Binance via le site Web ?**

1\. Connectez-vous à votre compte Binance et cliquez sur l’icône du profil, puis sur **\[Compte\]**.

![image](https://public.bnbstatic.com/image/cms/content/body/202403/4d1f2e0e900b88bc9fea339968a28222.png)

2\. Accédez à **\[Gestion des API\]** puis cliquez sur **\[Créer une API\]**. 

Veuillez remarquer qu’avant de créer une clé API, vous devez :

*   activer l’[authentification à deux facteurs (A2F)](https://www.binance.com/fr/support/faq/account-functions?c=1&navId=1#11)
     sur votre compte ;
*   effectuer un dépôt de n’importe quel montant sur votre portefeuille Spot pour activer votre compte ; et
*   vérifier votre [identité](https://www.binance.com/fr/support/faq/360027287111)
    .

![image](https://public.bnbstatic.com/image/cms/content/body/202403/05b8a1b8477ddcab3dc41fa67fadbaf9.png)

Pour en savoir plus sur l’API de rapports fiscaux, veuillez vous référer à [Comment obtenir un rapport fiscal sur Binance et questions fréquentes](https://www.binance.com/fr/support/faq/538e05e2fd394c489b4cf89e92c55f70)
.

3\. Sélectionnez le type de clé API de votre choix.

*   **Les clés API générées par le système** utilisent le chiffrement symétrique HMAC. Vous obtiendrez la clé API et la clé secrète.
*   **Les clés API auto-générées** utilisent le chiffrement asymétrique Ed25519 ou RSA. Vous recevrez une clé API, mais vous devrez créer votre propre paire de clés publique-privée via un logiciel tiers et fournir la clé publique à Binance.   
    Pour en savoir plus sur les clés API auto-générées, veuillez vous référer à [Comment générer une paire de clés Ed25519 pour envoyer des requêtes API sur Binance ?](https://www.binance.com/fr/support/faq/6b9a63f1e3384cf48a2eedb82767a69a)
    

![image](https://public.bnbstatic.com/image/cms/article/body/202311/47dd75a63be6a7bf78e192bd2c455b16.png)

4\. Saisissez un nom pour votre clé API. 

![image](https://public.bnbstatic.com/image/cms/article/body/202311/69a62f3856d38d4ae35324558ea88f65.png)

5\. Effectuez la vérification à l’aide de vos dispositifs A2F et de vos clés d’accès.

![image](https://public.bnbstatic.com/image/cms/article/body/202311/412eb2242603ab25a6c2b303f95aa9f2.png)

6\. Votre clé API est maintenant créée. 

**(Application) Comment créer votre propre clé API Binance via l’application ?**

1\. Connectez-vous à votre application de Binance et activez la version Binance Pro, puis cliquez sur **\[Plus\]** pour accéder au panneau Services.

![image](https://public.bnbstatic.com/image/cms/article/body/202406/7d025b8960e6192858e7cfc99c9065ca.png)

2\. Faites défiler jusqu’à la section **\[Autre\]** et cliquez sur l’icône **\[Gestion des API\]**.

![image](https://public.bnbstatic.com/image/cms/article/body/202406/114a3b66fdb4e24dde29ad3f0a832fcf.png)

3\. Cliquez sur **\[Créer une API\]**.

![image](https://public.bnbstatic.com/image/cms/article/body/202406/60fe2835ba5b47662e22af84512d024b.jpeg)

Veuillez remarquer qu’avant de créer une clé API, vous devez :

*   Activer l’[authentification à deux facteurs (A2F)](https://www.binance.com/fr/support/faq/account-functions?c=1&navId=1#11)
     sur votre compte ;
*   Effectuer un dépôt de n’importe quel montant sur votre portefeuille Spot pour activer votre compte ;
*   Vérifier votre [identité.](https://www.binance.com/en/support/faq/how-to-complete-identity-verification-for-a-personal-account-360027287111)
    

Pour en savoir plus sur l’API de déclaration fiscale, veuillez vous référer à la rubrique [Comment obtenir une déclaration fiscale sur Binance et questions fréquentes](https://www.binance.com/en/support/faq/538e05e2fd394c489b4cf89e92c55f70)
.

4\. Sélectionnez le type de clé API de votre choix. 

![image](https://public.bnbstatic.com/image/cms/article/body/202406/eadca4077a242b77e7a1d3f2085f04c8.png)

*   Les clés API générées par le système utilisent le chiffrement symétrique HMAC. Vous obtiendrez la clé API et la clé secrète.
*   Les clés API auto-générées utilisent le chiffrement asymétrique Ed25519 ou RSA. Vous recevrez une clé API, mais vous devrez créer votre propre paire de clés publique-privée via un logiciel tiers et fournir la clé publique à Binance.

Pour en savoir plus sur les clés API auto-générées, veuillez vous référer à [Comment générer une paire de clés Ed25519 pour envoyer des requêtes API sur Binance ?](https://www.binance.com/fr/support/faq/on-binance-6b9a63f1e3384cf48a2eedb82767a69a)

5\. Saisissez un nom pour votre clé API.

![image](https://public.bnbstatic.com/image/cms/article/body/202406/0b47dac4f851a1d35bdd6b19f08053df.png)

6\. Effectuez la vérification à l’aide de vos dispositifs A2F ou de vos clés d’accès.

![image](https://public.bnbstatic.com/image/cms/article/body/202406/880e4f21cf740eeb07b0c21d1bec99ca.png)

7\. Votre clé API est maintenant créée.

![image](https://public.bnbstatic.com/image/cms/article/body/202406/813f40ae74205afa5c2882df944fa18c.png)

### Questions fréquentes sur les clés API de Binance

#### 1\. Pourquoi ne puis-je activer aucune autorisation autre que la lecture ?

Nous vous recommandons vivement de ne pas activer les clés API pour des autorisations autres que la lecture sans définir les restrictions d’accès IP appropriées \[« Clé(s) API à accès IP non restreint »\]. La ou les clé(s) API HMAC à accès IP non restreint n’auront pas d’autres autorisations que celle de la lecture. Si vous souhaitez activer d’autres autorisations, veuillez ajouter des restrictions d’accès IP (au format IPv4), utiliser des clés auto-générées (telles que Ed25519 ou RSA) ou désactiver les contrôles de sécurité par défaut.

**Vue Web :**

![image](https://public.bnbstatic.com/image/cms/article/body/202311/3bbf8f45c328425f59e2af13040dff11.png)

**Vue de l’application :**

![image](https://public.bnbstatic.com/image/cms/article/body/202406/1542a6b4cb0a62687493f9676a0992a6.png)

#### 2\. Pourquoi ne puis-je pas activer l’autorisation de retrait ?

L’ajout de restrictions d’accès IP (au format IPv4) est obligatoire pour activer l’autorisation de retrait.

#### 3\. Pourquoi ma clé API ne peut-elle pas activer l’autorisation pour les contrats Futures ?

La clé API ne pourra pas activer l’autorisation pour les contrats Futures si elle est créée avant l’activation de votre compte Futures ou si vous avez activé la marge du portefeuille.

Pour se conformer aux dernières exigences du secteur, Binance a mis à jour ses services API. 

*   **Si votre IP n’est pas restreinte, l’autorisation de la clé API générée par le système ne pourra être définie sur \[Activer la lecture\] qu’à partir du 30/01/2023 à 00 h 00 (UTC).**
