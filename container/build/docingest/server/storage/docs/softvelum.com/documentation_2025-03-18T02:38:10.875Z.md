### Voice recognition

Live Transcoder is used in [AI-powered speech recognition](https://softvelum.com/nimble/speech-recognition/)
 based on Whisper.cpp library and model.  
Nimble Streamer allows automatically **generate closed captions** and **create translation subtitles** for live streams.  
The output is provided as WebVTT subtitles for HLS live streams. [Read more on the overview page](https://softvelum.com/nimble/speech-recognition/)
.
