![](https://custom-forms.gitbook.io/formify-docs/~gitbook/image?url=https%3A%2F%2Fcdn.formify.pro%2Fapi%2Fuploads%2Ftc81ik1f%2Fraw&width=768&dpr=4&quality=100&sign=cc714a43&sv=2)

#### 

[](https://custom-forms.gitbook.io/formify-docs/form-creation/formify-forms-via-website#begin-filling-in-your-form)

Begin filling in your form!

**Form Questions**

*   **Question** is required
    
    *   The question will be the actual question for when the user is filling out the form
        
    
*   **Min** is not required
    
    *   The min or minimum is the minimum length the users answer to the question needs to be, leave blank for none
        
    
*   **Max** is not required
    
    *   The max or maximum is the maximum length the users answer to the question needs to be, the default value is 1024
        
    
*   **Answer Body** is not required
    
    *   The answer body is the type of answer body will be used, the short one is one line while the paragraph body is multiple lines tall
        
    
*   **Placeholder** is not required
    
    *   The placeholder is text that will be shown inside the answer body before text is actually inputted, the text will become invisible after user starts typing
        
    

#### 

[](https://custom-forms.gitbook.io/formify-docs/form-creation/formify-forms-via-website#click-save-at-top-right-of-page)

Click "Save" at top right of page

![](https://custom-forms.gitbook.io/formify-docs/~gitbook/image?url=https%3A%2F%2Fcdn.formify.pro%2Fapi%2Fuploads%2Fvw9d6a9u%2Fraw&width=768&dpr=4&quality=100&sign=28e552d7&sv=2)

#### 

[](https://custom-forms.gitbook.io/formify-docs/form-creation/formify-forms-via-website#done)

Done!

You have created a form, it has been sent to your specificed channel! You can further edit it via the Form Editor.
