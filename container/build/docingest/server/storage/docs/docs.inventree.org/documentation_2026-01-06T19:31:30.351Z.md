# InvenTree - InvenTree Documentation

 

[Skip to content](https://docs.inventree.org/en/1.1.x/#features)

InvenTree - Intuitive Inventory Management
------------------------------------------

[Website](https://inventree.org/ "InvenTree Website")
 [Install](https://docs.inventree.org/en/1.1.x/start "Install InvenTree")
 [Mobile App](https://docs.inventree.org/en/1.1.x/app "InvenTree mobile app")
 [Translate](https://crowdin.com/project/inventree "Help translate InvenTree")
 [Source Code](https://github.com/inventree/inventree "Explore InvenTree source code")

InvenTree is an open-source inventory management system which provides intuitive parts management and stock control.

It is designed to be lightweight and easy to use for SME or hobbyist applications. Powerful business logic works in the background to ensure that stock tracking history is maintained, and users have ready access to stock level information. InvenTree is designed to allow for a flexible installation.

InvenTree is a [Python](https://www.python.org/)
 and [Django](https://www.djangoproject.com/)
 application which stores data in a relational database, and serves this data to the user(s) via a web browser, and (optionally) can be integrated into custom applications via an API.

Features[¶](https://docs.inventree.org/en/1.1.x/#features "Permanent link")

----------------------------------------------------------------------------

### Organize Parts[¶](https://docs.inventree.org/en/1.1.x/#organize-parts "Permanent link")

Parts are the fundamental element of any inventory. InvenTree groups parts into structured categories which allow you to arrange parts to meet your particular needs.

[Read more...](https://docs.inventree.org/en/1.1.x/part/)

### Manage Suppliers[¶](https://docs.inventree.org/en/1.1.x/#manage-suppliers "Permanent link")

InvenTree allows you to easily create, modify or delete suppliers and supplier items linked to any part in your inventory.

[Read more...](https://docs.inventree.org/en/1.1.x/purchasing/supplier/)

### Instant Stock Knowledge[¶](https://docs.inventree.org/en/1.1.x/#instant-stock-knowledge "Permanent link")

Instantly view current stock for a certain part, in a particular location, or required for an individual build. Stock items are organized in cascading locations and sub-locations, allowing flexible inspection of stock under any location. Stock items can be serialized for tracking of individual items, and test results can be stored against a serialized stock item for the purpose of acceptance testing and commissioning.

[Read more...](https://docs.inventree.org/en/1.1.x/stock/)

### BOM Management[¶](https://docs.inventree.org/en/1.1.x/#bom-management "Permanent link")

Intelligent BOM (Bill of Material) management provides a clear understanding of the sub-parts required to make a new part. InvenTree allows you to upload simple BOM files in multiple formats, and download a detailed BOM with all the information stored in its database.

[Read more...](https://docs.inventree.org/en/1.1.x/manufacturing/bom/)

### Build Parts[¶](https://docs.inventree.org/en/1.1.x/#build-parts "Permanent link")

InvenTree features a build management system to help you track the progress of your builds. Builds consume stock items to make new parts, you can decide to automatically or manually allocate parts from your current inventory.

[Read more...](https://docs.inventree.org/en/1.1.x/manufacturing/build/)

### Report[¶](https://docs.inventree.org/en/1.1.x/#report "Permanent link")

Generate a wide range of reports using custom templates. [Read more...](https://docs.inventree.org/en/1.1.x/report/report/)

### API[¶](https://docs.inventree.org/en/1.1.x/#api "Permanent link")

The core InvenTree software is implemented on top of a RESTful API, which can be used by external applications. Additionally, a native Python binding library is provided, for rapid development of programs to integrate with InvenTree.

[Read more...](https://docs.inventree.org/en/1.1.x/api/)

### Extend and Customize[¶](https://docs.inventree.org/en/1.1.x/#extend-and-customize "Permanent link")

InvenTree is designed to be highly extensible. If the core InvenTree functionality does not meet your particular need, InvenTree provides a powerful plugin system which can be used to extend on base functions as required.

[Read more...](https://docs.inventree.org/en/1.1.x/plugins/)
